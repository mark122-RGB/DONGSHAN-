## 〇、总纲：辩论的核心框架

### 0.1 辩论的本质

> **辩论的核心对象是人，目的是改变人。**

但人无法被别人直接改变——人只能被自己本就认同的事情重构平衡、自主选择改变。因此，辩论不是冷冰冰的逻辑模板，而是要在观众心里构建出一套"认识→取舍→确认"的逻辑，让听众通过共情和认同，主动站到我们这边。

**核心命题（新剑宗理论 — 李微中）**：辩论是立足事实，借用逻辑，指向价值的魔术。

```
事实  ←→  逻辑  ←→  价值
 ↑         ↑         ↑
论据      论证      持方意义
```

三者不可得兼，但好论者必在三者之间找到最佳平衡。

### 0.2 框架的三位一体：立场 + 方案 + 价值

任何一场高质量的辩论，框架都离不开三个核心模块，层层递进、相互支撑，且始终围绕"说服观众"展开：

**① 立场：明确"我们为谁而辩""主张什么"**

立场不只是简单的持方，而是要把辩题语言"翻译"成无歧义的"生活语言"，明确核心利益与理由，更要塑造一个具体的**"小明"**——他是你要为之发声的对象，是观众能共情的缩影。

> 示例：辩题"优绩主义是过时的教育理念"，我们的"小明"可以是"出身普通、资源匮乏，再努力也难突破阶层的高中生"，立场就可以翻译为"优绩主义'努力+天赋=资源分配'的逻辑，忽视了像小明这样的群体面临的资源不公"。

核心要求：态度明确一致，核心主张的论证完整，且所有观点都围绕"小明"的处境与诉求展开——让观众觉得你不是在空谈逻辑，而是在为真实的人发声。

**② 方案：提供"如何实现主张"的路径**

方案不是政策辩的专属，而是所有辩题打破僵局的关键。当双方都论证了自身的利弊，就需要通过方案强化利好、回避弊害，让"小明"的诉求有落地的可能。

方案可以是行为模式或政策倡导，无需复杂细节，但要贴合"小明"的实际。核心作用：让论证从"空谈定性"落地，让观众看到"我们的主张能真正帮到小明"。

**③ 价值：守住"主张背后的深层意义"，唤起共鸣**

价值是框架的"保底项"——即便事实战场僵持甚至处于劣势，仍能靠价值翻盘。它不是单纯煽情，而是通过连接"小明"的处境与更宏大的社会背景、理想期待，让观众认同我们主张的正当性。

> 示例：辩题"优绩主义是过时的教育理念"，价值可落脚"教育的核心是育人，不是筛选；当代社会更需要兼顾公平与人性的教育理念，让每个'小明'都有成长的机会"。

### 0.3 四大基础构成要素

核心模块需要具体要素支撑，这四个要素贯穿整场比赛，存在明确的逻辑关联：

```
定义（框定入口）
  ↓ 引导
讨论范围（限定边界）
  ↓ 限定方向
标准 / 零级论点（判断核心标尺）
  ↓ 拆分支撑
论点体系 / 一级论点（分维度论证）
```

**① 定义：框定讨论范围的"入口"**

定义的核心作用不是照搬字典，而是引导观众对辩题形成具体解读（即我方"小明"最关注的解读），进而框定讨论范围。

关键技巧：用语境带入定义，而非生硬罗列。可利用"人会直觉省略讨论范围"的原理。例如"解甲归田"辩题，反方可通过"战争未结束、个人有余力"的语境，自然框定讨论范围，而非单独给"解甲归田"下定义。

**② 讨论范围：限定"我们聊什么"**

讨论范围是定义的延伸，明确辩题的主体、场景、边界，避免跑题。它与定义、标准的关系：定义引导讨论范围；讨论范围限定标准的方向（同一讨论范围下可能有多个标准，但标准不能脱离讨论范围）。极端情况下，讨论范围可直接等价于标准。

**③ 标准 / 零级论点：判断持方成立的核心标尺**

标准是框架的"主心骨"，是判断我方立场成立的核心依据，本质是"零级论点"（最大、最上位的论点）。它不是一条简单的及格线，而是我方"小明"之所以坚定站在我方的最大理由，底层往往是"主体+价值"的结构。

- 单题制（如"中国应废除死刑"）：标准可围绕"需根解损"展开
- 双题制（如"人性本善/本恶"）：标准可设定为"哪种人性假设更能让小明这样的普通人实现良性成长"

标准的合理性需靠上位价值支撑。

**④ 论点体系 / 一级论点：支撑标准的分论点**

论点是对标准（零级论点）的拆分，用于分维度论证标准的合理性。每个一级论点在写作基础上需符合「论点句 + 双媒介 + 总结」的结构，且始终围绕「小明」的诉求展开：

- **论点句**：精炼概括观点，无需论证，仅为铺垫
- **媒介1**：推理与理由（因果论证或对比论证）
- **媒介2**：数据、案例、学理等现实背书
- **总结**：串联理由与证据，说明「这些证据能证明小明的困境，也能支撑我方标准」

> **基础标准 vs 美感追求**：上面四段式是写作的「骨架」——每一个论点在构思阶段必须清楚这四个部分各自是什么。到了实际成稿（见 §6.2），主张、推理、证据三者要在段落内融合成一段流畅的口播，不显式地拆成四块。**先有骨架，再求水乳交融。**

**论点拆分的三种方式**：

| 方式 | 结构 | 适用场景 |
|------|------|----------|
| **并集式** | 多个论点共同构成标准的全集 | 标准可拆分为并列维度 |
| **线性式** | 论点按逻辑链条递进 | 需因果链条推进的辩题 |
| **价值补强式** | 第一个论点支撑标准，第二个论点升华价值 | 需要兼顾逻辑与感动的辩题 |

### 0.4 框架的层级逻辑：价值阶梯

好的框架不仅有核心要素，还有清晰的层级逻辑——通过"价值阶梯"，可实现论点与标准的升降格。

**核心逻辑**：

```
上位抽象价值（不证自明的核心：公平、成长、幸福、尊严）
        ↑ 往上追"为什么重要"
标准 / 零级论点（判断持方成立的核心标尺）
        ↓ 往下拆"怎么实现"
下位具体事实 / 一级论点（论点句 + 双重媒介 + 总结）
```

- **往上追**：具体论点 → 标准 → 抽象价值（回答"这件事背后的深层意义是什么"）
- **往下拆**：抽象价值 → 标准 → 具体论点（回答"这件事具体能落地成什么"）
- **中间的"标准"**，是连接抽象价值和具体行为的核心枢纽

**阶梯搭建三步法**：

| 步骤 | 操作 | 方法 |
|------|------|------|
| **1. 定中间标准** | 先明确辩论的核心标准，作为阶梯的中间枢纽 | 问：我方持方成立的核心判断依据是什么？ |
| **2. 往上搭价值** | 追问"为什么这个标准重要"，直到触及不证自明的核心价值 | 标准 → 考上好高中 → 获得优质资源 → 实现个人潜能 → 上位价值：尊重每个学生的成长可能性 |
| **3. 往下拆事实** | 追问"这个标准怎么实现"，拆成3-4个具体的一级论点 | 上位价值 → 标准 → 具体事实1、2、3… |

**常见误区**：

| 误区 | 错误表现 | 纠正要点 |
|------|----------|----------|
| 层级无递进 | 下位=标准=上位，三者重复 | 每一级必须能回答"为什么"，不能重复 |
| 脱离"小明" | 价值空泛如"促进社会进步" | 阶梯必须围绕"小明"的处境与诉求 |
| 下位事实不具体 | "短视频能学东西"这种模糊表述 | 下位事实要具体到"行为、场景、对象" |

### 0.5 命题类型与框架适配

**单题制（非共存型论证，多为政策辩）**：

- 特征：仅一方能证成命题，逻辑终局是"政策应实行"或"不应实行"
- 正方框架构成（需根解损）：

| 要素 | 含义 |
|------|------|
| **需要性** | 现况有严重问题 |
| **根属性** | 问题成因与政策相关 |
| **解决力** | 政策能解决问题 |
| **损益比** | 政策利大于弊 |

- 反方挑战路径：①攻击合题性 ②维持现况 ③微调现况 ④提出相抗方案

**双题制（共存型论证，多为价值辩）**：

- 特征：双方命题可同时为真，核心争夺"定义、讨论范围与标准"
- 攻防关键：争夺定义与标准的采纳理由 → 在己方框架下完成"论点-论据"闭环
- 评委判断逻辑：先看双方是否提出完整框架 → 再看排除理由 → 最后检视关联性

**双题制快速诊断：和解空间测试**

在破题阶段，先问一个前置问题：**双方有没有"和解空间"？**

> 和解空间 = 双方核心主张可以同时成立、不存在必然冲突的区域。

如果存在和解空间，那么**这场比赛必然是定义战**。因为不存在实质分歧，胜负完全取决于"谁来定义核心词"——谁的定义被采纳，谁就赢了。

诊断流程：
1. 写出双方核心主张（各一句话）
2. 问：两者能同时为真吗？
3. 若能 → 这场比赛是定义战 → 胜负取决于定义与标准的争夺
4. 若定义战 → 谁的框架能同时实现双方主张，谁就拥有结构性优势

> 例：「当代年轻人应该活得更现实/更理想」——正方"实事求是的方法论"与反方"追求真和善"并不必然冲突。故这场比赛从头到尾是定义战。正方"方法论"战场能同时容纳双方主张，反方"目的"战场却无助于方法论选择——正方结构性占优。

## 一、破题与框架搭建（赛前 + 一辩）

> **核心任务**：搭建完整框架（定义+讨论范围+标准+论点体系+方案+价值），塑造"小明"形象。

### 1.1 破题：新剑宗立论四问（李微中）

在搭建框架之前，先用四问锁定方向：

| 步骤 | 问题 | 目的 |
|------|------|------|
| 1 | 这个题实际上是在问什么？ | 寻找合适的可证伪处——切点 |
| 2 | 我方需要论证什么？ | 负担举证义务——怎么切 |
| 3 | 我方怎样才能完成论证？ | 寻找论证的方式与标准 |
| 4 | 这样理解对人有何意义？ | 人应该选择怎么行动 |

### 1.2 论的定义与好论的条件

> **论，是让某一持方得以成立的充分理由。**

- 持方不是事实判断，而是价值判断——价值判断的关键，是要对人产生意义
- 不同的持方对应着不同的世界 → 不同持方下，人会有不同的行为
- **定理**：如果论不同，人的行为不会完全相同；论必须具有可证伪性

**好论的五个条件**：

1. 会让人的世界观有所变化，并为之作出改变
2. 在确保攻防优势的前提下，具有可证伪性
3. 明确自己所需要负担的论证义务
4. 以某种方式负担之，达到足以抵消怀疑的程度
5. 论证义务完成 = 可证伪处未被证伪 = 充分理由

### 1.3 塑造"小明"：从抽象辩题到具体的人

"小明"是你要为之发声的对象，是观众能共情的缩影。塑造要点：

- **具体化**：给"小明"一个明确的身份、处境、困境
- **典型化**：他要能代表辩题涉及的核心群体
- **共情化**：让观众能感知小明的痛苦、渴望与诉求

> 示例：辩题"短视频对青少年利大于弊"，小明可以是"喜欢刷短视频、缺乏自控力但充满好奇心的初中生"。

### 1.4 定义与讨论范围的确立

**定义的核心技巧**：
- 用语境带入定义，而非生硬罗列字典释义
- 定义要为后续的标准和论点服务——要能自然引导出我方的讨论方向
- 可利用"人会直觉省略讨论范围"的原理，不单独给定义，而是用语境框定

**讨论范围的确立**：
- 明确辩题的主体、场景、边界
- 排除极端案例（既有天赋异禀又完全不努力的人）
- 讨论范围与标准的关系：讨论范围限定标准方向

### 1.5 标准的确立与论证

标准是我方框架的"主心骨"，底层结构通常是"主体+价值"。

**确立标准的方法**：
1. 明确"小明"最关注什么
2. 将"小明"的关注点翻译为可判断的标尺
3. 用上位价值论证标准的合理性

**驳斥对方判准的三个角度**（张子龙）：

| 角度 | 含义 |
|------|------|
| **合理性** | 这个判准是否符合常识？ |
| **可辩性** | 这个判准是否让辩题失去了讨论空间？ |
| **启发性** | 这个判准是否能让讨论更有深度？ |

### 1.6 立论撰写：从常识中来，到常识中去（李微中）

**立论应当包含**：
- **完整的逻辑链**：诠释辩题想干嘛 → 如何讨论辩题（双方应论证什么）→ 我方如何论证 → 补价值
- **事实与例子**：情景化解释 → 复杂概念生动化 → 论点佐证强化 → 通俗化
- **主次分明**：关键处 > 复杂处 > 会被挑战处 > 其他地方

**立论的核心目标——一个让评委懂但不要让对手懂的立论**：

| 原则 | 技巧 |
|------|------|
| 用浅显的论点预示深刻的内涵 | 实际理由必须通俗易懂，但选择这些理由大有玄机 |
| 用公允的讨论隐藏攻防的野心 | 立论过程要公允得不能再公允，但讨论方向指向己方攻防优势 |
| 用核心的论述辐射整个世界 | 只讲明白一件事，讲得通透而周全；之后再看待整个世界的任何质疑，无不在此论述之下 |

### 1.7 定义战：从世界观到价值观的完整操作链

> 定义战不等于"下定义"。定义战是一场比赛中决定胜负的战略形态——定义一旦站稳，进可攻退可守；定义一旦被拆，整个立论随之坍塌。

**定义的双层结构**：

| 层次 | 核心问题 | 标准 | 辩论操作 |
|------|----------|------|----------|
| **世界观** | "能不能这样定义？" | **不矛盾**——前后一致，能解释所有案例 | 归谬法起手：用符合对方定义但极度违背直觉的例子，逼对方要么认怂改口，要么硬着头皮解释 |
| **价值观** | "为什么要这样定义？" | **不违和**——符合人们的道德直觉与生活常识 | 建立理论说明"这样定义之后，我们能看到什么更有意义的东西" |

**世界观层的攻防**：

- 攻击方：通过归谬让对方陷入两难——坚持定义=违背直觉；放弃定义=定义战认输
- 防守方：只要做到不自相矛盾（同一类型的案例用同一标准评价），世界观层就能守住
- 常见错误：攻击方只攻世界观（"你这个定义好奇怪"），从不追问价值观（"你为什么非要这么定义"）→ 即使归谬成功，也只是让对方难堪，没有真正拆掉

**价值观层的攻防**：

- 进攻方必须追问："就算你能这么定义，你为什么要这么定义？凭什么你的定义比常识更优？"
- 防守方必须回答：这个定义让我们看到了______（常识看不到的东西），而这个东西之所以重要，是因为______
- 正方常见的价值观论证策略：我的定义是一种"方法论"（怎么做事），对方的定义是一种"目的论"（做什么事），方法论比目的论更根本——因为好的方法论能帮你实现任何目的，而只知道目的却不知道方法，可能南辕北辙

**定义战中的角色分工**：

| 环节 | 任务 |
|------|------|
| 一辩申论 | 给出定义，建立"世界观"的解释框架（能这样定义），铺垫"价值观"（这样定义的意义） |
| 被质询时 | 守住"世界观"——面对归谬，态度坚定，不自相矛盾 |
| 质询对方时 | 归谬起手→逼出对方世界观的矛盾→追问价值观 |
| 自由辩 | 用我方定义反复解释对方的例子（"你举的这个，恰恰是我方立场的体现"），展示定义的解释力 |
| 结辩 | 总结两种定义代表的两种世界观，说明我方定义更优的价值观理由 |

**定义战的致命时间窗口**：

定义战**必须在对方用自己的定义开始收割之前打掉**。一旦对方用定义拆完了你方所有论点，再来攻击定义为时已晚——裁判已经用对方的框架理解完整场比赛了。

## 二、攻辩交锋（二、三辩）

> **核心任务**：攻防框架漏洞，强化己方核心要素，维护"小明"的合理性。

### 2.1 质询体系（张子龙）

#### 2.1.1 质询的本质

> 质询的本质不是辩论，是对话。对方可以即时回答，因此质询的操作分为三步。

**质询三步法**：

```
第一步：定共识
    ↓
第二步：打问题（逻辑的矛盾 / 事实的不符）
    ↓
第三步：给总结——告诉大家你哪里有问题，我哪里对
```

**第一步 · 定共识**：先把对方的主张、前提、结论确认下来。不确定的东西不打——打错了浪费自己的时间，打歪了给对手留退路。

确认框架：
```
论点 → 前提 → 结论
  ↓       ↓       ↓
有无？  有无/强度？ 证成命题？
                   其他可能/内部矛盾？
```

**第二步 · 打问题**：共识确认之后，指出对方逻辑上的裂缝或事实上的不符。这一步和「确认」不冲突——先确认了再打，打的是确认到的东西。

**第三步 · 给总结**：把上面两步的产物收束成一句话。「对方辩友你主张的是 X，但 X 的问题在于 Y——所以你这个点不成立。」

#### 2.1.2 质询的黄金原则

**① 确认、确认、再确认**

核心追问路径：
1. 他的论点和前提是什么？他有证明吗？他的证明有说服力吗？
2. 他的前提真的可以推出结论吗？
3. 他的结论真的有论证命题吗？

**② 学会问"为什么"**

连续追问，直到逻辑链条彻底暴露：
> 为什么跳槽不利于成长？
> → 因为青年人太自我中心
> → 为什么太自我中心？
> → 因为想法和社会不一样
> → 为什么这样就是自我中心？
> → 这样才能归因为自己，归因自己才有利于成长
> → ……（循环追问）

#### 2.1.3 质询的节奏与分寸

| 技巧 | 说明 |
|------|------|
| **打断时机** | 表达足够精确时才可打断，宁可慢半拍不抢一瞬 |
| **见好就收** | "没有趁胜追击这种事情，你原本打算问什么，就问什么！" |
| **黄金时长** | 每套问题控制在30-45秒，预算30-45%时间用于控场和应对干扰 |

**追问纪律：对方没正面回答时，至少再问两次**

比赛中常见一种致命失误：质询方问到了关键问题，对方换了个说法搪塞过去（"计划本身是活得现实的表现"），质询方却没有追打、自己转移了话题。

识别信号：
- 对方回答了一句话，但**意思等于原问题中的某个词换了说法**
- 对方用"你方的定义是……"反推，而不是正面回答你的问题
- 对方答非所问，开始举例子

应对纪律：
1. 识别到对方没正面回答 → 立刻重复原问题（可换一个问法，但同一指向）
2. 对方再次回避 → 第三次追问，并可直接点出："你没有回答我的问题，我的问题是______"
3. 对方三次回避 → 记录在案，转给后续环节："对方辩友刚才三次没有正面回答______，请评委记录"

> 注意：追问不等于纠缠。如果对方给出的是一个实质回答（即便是对你方不利的回答），就应该接住而非机械重复。追问纪律只针对"换说法搪塞"的情况。

#### 2.1.4 四种"无法回答"的应对话术

| 对方回应类型 | 破解方法 |
|-------------|----------|
| 归谬型恶逃 | "好嘛，不然你觉得人们是怎么想的呢？" |
| 派别型恶逃 | "我怎么知道别人怎么想"→ 切换为"你方的主张是什么？" |
| 弃攻型恶逃 | 直接指出对方回避，记录在案 |
| 自洽型顽固 | "你说的有道理，但这不能证明你的持方" |

### 2.2 接质体系（柴子凡）

#### 2.2.1 接质的根本目标

**守住己方的论证体系**。完整的论证体系有三层：
1. **事实前提**：排除合理怀疑（注意变量改变——客观条件、制度环境变迁）
2. **价值底线**：足够的素材与空间（价值底线无法依靠事实逻辑证成）
3. **逻辑推理**：强度契合、形式有效（演绎推理 + 当然推理）

#### 2.2.2 接质六项实用技巧

| 技巧 | 要点 |
|------|------|
| ① 回答优先级 | 实质内容 > 形式标准 > 空话套话 |
| ② 省略/缩短/后置让步 | 防止"话还没讲完就被打断" |
| ③ 答"超"所问 | 主动推进讨论进程，承认底线并渲染价值 |
| ④ 用例子代替逻辑 | 抽象逻辑链用直观例子替代 |
| ⑤ 虚例语境补全 | 主动补全对方虚例中未说明的前提 |
| ⑥ 警惕诱导发问 | 识别隐藏的未经确认的前提 |

### 2.3 反驳体系

#### 2.3.1 论证三要素

```
论点 Claim   — 核心观点，须满足：明确性 · 可辩性 · 相关性
论据 Evidence — 支撑血肉，须满足：权威性 · 时效性 · 代表性
论证 Warrant — 逻辑桥梁：演绎 · 归纳 · 类比
```

#### 2.3.2 反驳的四个维度

| 维度 | 核心 | 方法 |
|------|------|------|
| **一：反驳定义** | 釜底抽薪，重划战场 | 拆解概念边界 → 重构有利定义框架 |
| **二：反驳逻辑** | 斩断链条，识别谬误 | 识别稻草人/滑坡/以偏概全/因果混淆 |
| **三：反驳论据** | 去伪存真，质疑支撑 | 质疑真实性 + 相关性 + 充分性 |
| **四：反驳价值** | 揭示偏颇，升维打击 | 剖析对方价值预设 → 提出更优普世价值 |

#### 2.3.3 反驳四层次（黄执中）

| 层次 | 名称 | 表现 | 评价 |
|------|------|------|------|
| 1 | 反驳观点 | 仅说"我不同意"，无实质论据 | 最低级，情绪宣泄 |
| 2 | 反驳逻辑 | 指出逻辑谬误与推理漏洞 | 合格，能有效削弱 |
| 3 | 反驳前提 | 攻击对方隐藏的未言明假设 | 高级，釜底抽薪 |
| 4 | 反驳价值 | 上升至价值层面，提出更优解 | 最高级，破立兼具 |

#### 2.3.4 常见逻辑谬误速查

| 谬误 | 描述 | 经典例句 |
|------|------|----------|
| **稻草人** | 歪曲对方观点再攻击 | "如果你不支持这政策，你就是不爱国" |
| **滑坡谬误** | 夸大行动的连锁后果 | "今天允许同性婚姻，明天人和动物结婚" |
| **以偏概全** | 个别案例推普遍结论 | "比尔·盖茨辍学成首富，所以高教无用" |
| **因果混淆** | 相关性误作因果 | "冰淇淋销量上升→溺水事故增多，所以冰淇淋导致溺水" |

#### 2.3.5 高级反驳战术

**归谬法**：假设对方观点为真 → 顺着逻辑推导 → 得出荒谬结论 → 反证原观点错误

**借力打力**：承认对方部分合理性 → 利用其逻辑前提 → 证明这恰恰支持己方观点

**顺水推舟**：不否定对方前提 → 顺着逻辑延伸 → 导出对方无法接受的结论

**反驳强度等级**：
- 最弱：质疑（"你是不是不对？你怎么证明？"）
- 最强：击穿（拆掉："你这样不对，因为……" / 吃掉："你对了也没用，因为……"）

#### 2.3.6 临场反驳四步法（谢宇·定列比结）

| 步骤 | 动作 | 含义 |
|------|------|------|
| **定** | 定点 | 明确反驳对象——你要打什么？ |
| **列** | 陈列 | 复述双方观点——你说的 vs 我说的 |
| **比** | 比较 | 分出高下优劣——为什么我更好？ |
| **结** | 联结 | 指出立场关联——这与我们的持方什么关系？ |

#### 2.3.7 反反驳标准化模板

> 对方反驳你方论证时，通常有三种攻击方向。每种都有标准化的回应套路。

**对方说"A未必有B"（你的方法论未必导致你主张的好结果）**

| 回应 | 话术模板 |
|------|----------|
| **"没有A会更糟"** | "你举的例子如果按照我方方法论去做都这样了，那不按我方方法论去做的结果只会更差。" |
| **"不是A有问题，是做A不够好"** | "这恰好说明他还没有真正做到我方主张的方法论——如果他做得更好，就不会是这个结果。" |

**对方说"没有A也有B"（不按你的方法论也能成功）**

| 回应 | 话术模板 |
|------|----------|
| **"这是幸存者偏差/运气"** | "不按我方方法论也能成功的案例，是运气好——更多的同类案例失败了，只是你方没有举出来。" |
| **"有A会更好"** | "不按我方方法论都能做到这个程度，如果按了，岂不是能做得更好？" |

**对方说"A会导致C这个坏处"（你的方法论会导致某个恶果）**

| 回应 | 话术模板 |
|------|----------|
| **"这是别人介入造成的，不是A造成的"** | "导致坏后果的不是我方主张——是他人介入的行为。你不能把别人的错算在我方的头上。" |
| **"对方替代方案导致C的风险更大"** | "每一种选择都有风险。问题是，我方的方法论导致C的风险，比对方方案导致C的风险更小。" |

> 以上模板的核心思路：不否认对方指出的现象，而是**重新归因**或**升维比较**。这些不是哲学武器，只是反驳中的因果归因训练——辩手需要在模辩中反复练习，直到成为条件反射。

## 三、自由辩论（全员）

> **核心任务**：深化框架分歧，巩固己方优势，强化"小明"的诉求。

### 3.1 战场的两种类型（卢卡斯）

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| **推进主导型** | 自闭推进己方观点，呈现优势，常使用歌词模式 | 己方立论占优时 |
| **攻防主导型** | 重点在交锋上击溃对手，不封闭，与对手交流性高 | 需要撕开对方防线时 |

**战场之间的"互动关系"——决定谁能同时吃下双方利益**

一场比赛往往不只一个战场。审局的关键是判断**战场之间有没有互动关系**：

> 互动关系 = 甲方战场成立会对乙方战场产生影响，反之则无。

| 互动类型 | 含义 | 策略价值 |
|----------|------|----------|
| **单向影响** | A战场成立有助于B战场（如"方法论"有助于"实现目的"），但B战场成立对A战场无帮助 | 拥有A战场的一方结构性占优——因为他可以同时实现双方的主张 |
| **双向无关** | 两个战场互不影响 | 谁在两个战场上都占优谁赢；各赢一个则平局，比损益比 |
| **互相冲突** | A战场成立意味着B战场不成立，反之亦然 | 定义战的典型形态——争夺定义就是争夺哪个战场被计入胜负 |

**拥有互动关系优势的一方，最安全的打法是**：稳住自己的定义；收割所有的利益；用自己的定义拆解对方的一切论点。

> 例：正方"方法论"战场成立后，可以直接说"你方的理想很好，所以要用我方实事求是的方法论来实现这个理想"——正方同时吃下双方利益。反方"目的"战场即便成立，也无助于告诉年轻人具体该怎么做事。

### 3.2 战场的"起承转合"结构

```
起（阐明战场重要性）
  → 承（补充更多资料，输出伤害）
    → 转（做比较——拿对方劣势比己方优势）
      → 合（为战场做总结）
```

### 3.3 自由辩操作核心：Blocking（挡）

> 不是反驳，而是堵住对方的口，让你可以继续推演战场。

具体技巧：提相抗、追加责任、区隔缩小、顺手转。

### 3.4 三种攻防一体高阶招法

| 招法 | 公式 | 例 |
|------|------|-----|
| **前提型战场** | "要讨论XXX，就必须先讨论YYY" | "要讨论控枪的好处，就必须先讨论控枪的过程" |
| **内涵型战场** | 重新定义核心概念的内涵 | "拥有枪支是美国文化，但有害的文化也要传承吗？" |
| **盖罪名** | 提高对方论证成本 | "对方辩友您现在要做的是……" |

### 3.5 自由辩节奏控场（陈劭康）

- 以**4个战场**为基本规划（4分钟自由辩）
- 每个战场耗时1分钟，4-5个发言次
- 每个发言次控制在**12-15秒**
- 优先顺序：先争定义与讨论范围 → 再争标准 → 最后争论点

**团队控场三要素**：
1. **整体流动意识**：赛前统一战场顺序和负责人
2. **全程沟通**：像电竞/篮球，全程口头沟通
3. **中间辩位是沟通桥梁**：上传下达

#### 3.5.1 临场指挥四能力：听说读写

场上必须至少有一人能同时做到以下四项——这不一定是四辩，也不一定是队长，而是"团队中审局最快的那个人"。

| 能力 | 含义 | 具体操作 |
|------|------|----------|
| **听** | 迅速听懂场上形势 | 对方这个反驳瞄准的是我方哪个位置？是打定义还是打逻辑？ |
| **说** | 整理比赛型发言 | 把审局结论转化为简短指令："守定义""转战场""用运气打回去" |
| **读** | 看懂队友的记录 | 四人必须共用一套场上记录符号/体系 |
| **写** | 记清比赛形势，让队友看懂 | 关键攻防节点画图记录，不能只靠脑子记 |

**场上记录体系（最低共识）**：

| 记录要素 | 格式 |
|----------|------|
| 对方核心论点 | 编号+一句话（如"①理想=原则判断"） |
| 我方已拆标志 | 打钩（✓已解决）或打叉（✗待处理） |
| 对方未回应的我方追问 | 星号+内容（如"*对方未回答'为什么理想者不能有计划'"） |
| 战场状态 | 简短标注（如"定义战：我方优势""目的战场：平"） |

#### 3.5.2 审局与战场制图

> 审局不是在结辩时才做——审局是贯穿全场的连续动作。每过一个攻防环节，场上形势就更新一次。

**制图格式**（每次攻防环节结束后更新）：

```
┌─────────────────────────┐
│ 战场一：定义（方法论）      │
│ 正方：活得现实=实事求是   │
│ 反方攻击：葛朗台/陶渊明    │
│ 状态：正方守住（←暂未追问价值观）│
├─────────────────────────┤
│ 战场二：目的（追求什么）    │
│ 反方：求真求善/自由人格    │
│ 正方攻击：用实事求是更能实现│
│ 状态：正方拆解成功         │
└─────────────────────────┘
```

**制图判断依据**：

| 战场状态 | 标志 |
|----------|------|
| **一方优势** | 该方定义/论据对方未有效回应，或每次回应都被反拆 |
| **平局/未定** | 双方均有有效攻防，未分出明显高下 |
| **潜在威胁** | 对方问到了关键但未追打，随时可能被后续环节重提 |

**审局结论直接影响战术决策**：
- 若定义战场我方占优 → 命令全队"一切回归定义"，用定义打对方一切举例
- 若定义战场受威胁 → 优先在下一个环节加固定义，暂缓推进新论点
- 若对方某核心论点仍未被我方触及 → 指定下一轮质询/自由辩专门处理

### 3.6 对辩节奏三种模式

| 模式 | 操作 | 时间效率 |
|------|------|----------|
| **先守后攻** | 先回答再提问，最常用 | 两步操作，耗时较多 |
| **借力打力** | 用对方词汇组装问题问回去 | 一步操作，高效高技 |
| **自由变化** | 忽长忽短，让对方无法习惯 | 出其不意 |

## 四、结辩升华（四辩）

> **核心任务**：梳理框架交锋，强化己方优势，升华"小明"背后的价值。

### 4.1 结辩的核心定位

结辩是整场比赛的"收口"，不是重新立论、不是继续攻防，而是：
1. **梳理分歧**：明确双方框架的核心差异及攻防结果
2. **强化框架**：重申定义、讨论范围、标准与论点的闭环，维护"小明"的合理性
3. **价值升华**：结合社会背景，说明"支持小明"的深层意义

### 4.2 结辩操作要点

**该做的**：
- 直面核心争议——对方追打什么就回应什么
- 淡化逻辑/事实，回归价值与情感——这是结辩最强大的武器
- 展现气场与诚意，说"其实今天的比赛很简单"
- 把具体争议拉升到"这对那群人意味着什么"——让他们不再是辩题里的抽象概念，而是评委能感知到的人

**不该做的**：
- 不重复队友——不做"我方二辩举证的XX数据对方没回应"的技术性归纳
- 不装作若无其事——有明显劣势时不回避
- 尽量不做前置攻防——结辩还在处理定义和判准，基本翻不了

**各环节的时间窗口纪律**：

| 环节 | 最晚必须处理的事 | 过了这个环节没处理=基本来不及 |
|------|-----------------|-------------------------------|
| **反一被质询后** | 识破正方是否在打定义战 | 若到自由辩才意识到，时间不够 |
| **反二申论** | 攻击正方定义（若正方靠定义吃饭） | 正方用定义拆完你方论点后再打定义，评委已用正方框架理解全场比赛 |
| **正三/反三小结** | 锁定胜负手，放弃已死战场 | 还在同时打三个战场=一个都赢不了 |
| **自由辩前半段** | 巩固前三个环节建立的框架优势 | 自由辩后半段只能收割，翻框架需要时间 |
| **结辩** | 放弃一切未解决的纯逻辑攻防，只做价值收束 | 结辩还打定义战=本场框架战已经输了 |

### 4.3 幽默与感动

> "幽默是最好的防守，感动是最好的进攻。"

结辩的语言要能直击人心——不是华丽辞藻的堆砌，而是回到破题时小明所代表的那些人：是城中村里每天通勤四小时的年轻人，是付出了全部努力却被说「你还不够拼」的普通人。让评委和观众在最后时刻感受到：我们不是在打一场逻辑游戏，我们是在为一个真实的人的处境而战。

## 五、专项能力

### 5.1 价值渲染（梁秋阳）

#### 5.1.1 核心公式

> **细节 → 真实 → 冲突 → 认同**

辩论的深层竞争是心智占领：
```
事实资讯 → 逻辑推论 → 观点共鸣 = 胜利
```

#### 5.1.2 价值渲染的三种进路

| 进路 | 方法 | 关键要点 |
|------|------|----------|
| **戏剧冲突** | 找到"相反与相对"，制造价值张力 | 公平的反义词不只是"不公平"，还可能是"效率" |
| **细节** | 把抽象描述变为具体枚举 | "他带着一只猴子、一只猪、还有一个河里的妖怪，走上十几年" |
| **真实（Keep it real）** | 直接 > 间接，私人心声 > 公共文本，断言 > 推论 | 用你自己的话，说真实的感觉 |

#### 5.1.3 价值冲突制造法

**找到价值的"反面"**：
> 正义 → 不公（相反） → 不义（矛盾） → 专制（负面之负面） → 稳定

每深入一层，价值的张力就增强一分。

**低打与高打**：
- **低打**："为什么"问句引导 → 曲径通幽
- **高打**："凭什么"犁庭扫穴 → 酣畅淋漓

#### 5.1.4 价值渲染实战模板（以"自由"为例）

| 版本 | 文案 | 效果 |
|------|------|------|
| 初始 | "自由是人类的天性，我们渴望自由" | 虚空，无共鸣 |
| +冲突 | "即使有100%安全的保证，代价是从此没有隐私，可以吗？" | 制造两难，引发思考 |
| +细节 | "代价是你每一秒都活在监察和控制之下" | 画面感增强 |
| +真实 | "我问你：如果保证你和家人再也不会受到伤害，代价是你必须每一秒都活在监察和控制之下，你会接受吗？……还是说，你也知道，在我们每个人心底里都还有那一分对自由的向往。" | 直击心灵 |

### 5.2 语境塑造（张子龙）

#### 5.2.1 语境塑造三层次

| 层次 | 方法 | 效果 |
|------|------|------|
| **1 增加细节** | 强化旧的语境 | 让原有框架更鲜活有力 |
| **2 改变脉络** | 建立新的语境 | 在具体与抽象之间切换 |
| **3 转换框架** | 塑造新的认知 | 给观众一副新的"眼镜" |

#### 5.2.2 核心工具：上堆下切法（NLP）

**上堆（抽象化）**：把具体问题升维
> 创业失败 → 学业和爱情失败 → "失败"本身 → 人生挫折

**下切（具体化）**：把抽象概念降维
> 财富 → 资产 → 农产 → 家畜 → 母牛 → 阿花（那只具体的牛）

#### 5.2.3 辩题框架重构三步

**步骤一：重设战场**（"这个题目有趣的部分不是……而是……"）
> 例：爱上人工智能算不算爱情 → 这个题有趣的部分不是爱情，而是当科技越来越进步，虚拟和现实还有没有差别？

**步骤二：铺陈战场**（用一个故事/电影/现象展开新视角）
> 例：《黑客帝国》Matrix，红药丸与蓝药丸的选择

**步骤三：价值抬高**（从具体情境升维到人类根本命题）
> "因为我们还在意真假，不是因为真实比较幸福，不然，没尊严！"

### 5.3 资料运用（唐梦）

#### 5.3.1 备赛资料四分类

| 类型 | 内容 | 用途阶段 |
|------|------|----------|
| **背景资料** | 题解描述、辩题来源、题设背景 | 解辩题 |
| **参考范例** | 参考辩案、论点、辩词、视频 | 头脑风暴 |
| **定义** | 专业术语定义、一般名词定义、合题案例 | 立论建构 |
| **论据** | 统计数据、调查报告、论文结论、专家证言 | 立论建构 |

#### 5.3.2 资料检索渠道

| 渠道 | 具体来源 |
|------|----------|
| 政府统计 | 统计公报、统计数据 |
| 商业报告 | 招股说明书、行业协会报告、券商行研报告 |
| 数据库 | 知网、万方、维普、北大法宝、WestLaw、国泰安 |
| 一手资料 | 电话访谈（政府/行业协会/企业）、问卷调查 |

#### 5.3.3 不同类型辩题的资料侧重

| 辩题类型 | 定义依赖度 | 论据依赖度 | 资料重点 |
|----------|-----------|-----------|----------|
| **政策类** | 低 | 高 | 数据、研究结论、专家证言 |
| **时事/商业类** | 较低 | 较高 | 学术理论（社会学/心理学/经济学/管理学） |
| **价值思辨类** | 高 | 低 | 合题案例的举例与阐释 |
| **脑洞类** | 较高 | 较低 | 辩题假设在现实中的逻辑推论 |

#### 5.3.4 资料拆解三板斧

| 方法 | 操作 | 杀伤力 |
|------|------|--------|
| **可靠性拆解** | 质疑样本容量、时间、方法、利益相关 | 适用于几乎一切论据，但只能削弱不能排除 |
| **关联性拆解** | 逻辑链拆解、类推归谬 | 杀伤力最强，可彻底破坏论据效力 |

#### 5.3.5 比赛中呈现资料的技巧

- **节选**：不应通篇念读，择选重点
- **修辞**：不应照搬原文，加以修辞
- **解读**：呈现资料后需注重解读
- **自由辩3秒原则**：资料呈现以3秒（约20字）为限；不问对方"知不知道"，直接提数据诘问

#### 5.3.6 举例前的硬核核实义务

> 一场比赛中，最致命的打脸不是逻辑被拆，而是**你举的例子恰恰是对方立场的例证**。当你用新闻事件、历史案例来支撑论点时，必须过一手信源，而不是依赖记忆或标题。

**核实清单（每个例子出稿前必答）**：

| 核实项 | 操作 |
|--------|------|
| **事实时间线** | 事件发生的准确时间、顺序——"赛后拒唱国歌"和"赛前拒唱"是完全不同的事 |
| **当事人一手说法** | 当事人自己怎么说的？有没有采访/手记/自述？优先一手信源，忌转述 |
| **因果方向** | 你自己认定的"因为理想所以做了X"——当事人真的是出于这个原因吗？还是有更复杂的综合考量？ |
| **结果确认** | 结果是成功了还是失败了？有没有被后续事件反转？ |

**常见信源层级**：

| 层级 | 信源 | 可信度 |
|------|------|--------|
| 一手 | 当事人采访/手记/自述、原始文件、官方公告 | 最高 |
| 二手 | 正规媒体报道（路透/新华社/NYT等）、学术论文 | 较高，但需交叉验证 |
| 三手 | 公众号/自媒体/百科/网友总结 | 仅作线索，不可直接当作场上论据 |

**致命红线**：
- **不要只看标题和简介就引用新闻事件**。读完全文，确认因果方向。
- **不要说"赛后"而不核实**。时间顺序错=当场被对方用事实打脸。
- **不要把复杂的政治/社会事件简化为"某人出于XX理想做了XX"**。真实世界中的决策远比动机单一的叙事复杂。

> 例：「伊朗男足拒唱国歌」——反方用此例支持"活得理想"：球员出于理想拒唱国歌。但一手信源显示：①这是赛前而非赛后；②球队2:6惨败；③球员采访称"从未过分关注此事"；④下一场比赛在当局威胁家人后立即恢复唱国歌。这个例子更接近"多方势力博弈下的现实权衡"，恰恰是正方立场的例证。

## 六、内容输出格式规范（订制包模板）

> **设计理念**：§〇至§五是「怎么想」（头脑风暴），本节是「怎么输出」（格式框定）。
> 以下 8 个模块构成一套完整的**辩论订制包**，每个辩题按此模板全量产出。

---

### 6.1 订制包总览

> 模块按**比赛实际流程**排列。备赛时也建议按此顺序产出——每一环节的输出直接对应场上该环节的需求。

| 序号 | 模块 | 对应环节 | 核心功能 |
|------|------|----------|----------|
| **①** | 参考一辩稿 | 一辩立论 | 完整立论陈词，3分钟口播体 |
| **②** | 四质一 | 四辩质询对方一辩 | 质询对方一辩的问题链 + 拆解对方一辩框架 + 我方一辩接质防守 |
| **③** | 二辩申论稿 | 二辩申论 | 反驳对方框架 + 延伸己方一个论点 |
| **④** | 三质二 | 三辩质询对方二辩 | 质询对方二辩的问题链 + 拆解对方二辩申论 + 我方二辩接质防守 |
| **⑤** | 三辩小结 | 三辩小结 | 收质询 + 补框架 + 做比较 + 给决胜点 |
| **⑥** | 自由辩攻防要点 | 自由辩论 | 战场规划 + Blocking话术 + 攻防节奏 |
| **⑦** | 结辩稿 | 四辩结辩 | 争议梳理 + 价值回调 + 情感升华 |
| **⑧** | 资料卡 | 全员备赛（贯穿全场） | 表格化论据库，含数据+话术+攻防 |

---

### 6.2 模块①：参考一辩稿格式

**结构模板**：

```
谢谢主席，各位好。

[第一段：定义框定与语境建立，100-150字]
从辩题核心词切入，建立一个能让听众重新理解辩题的框架。可以援引思想家/理论作为定义支撑（如"黑格尔说，现实即是一切事实的总和"），但不要生硬罗列字典释义。关键在于把辩题翻译成"我方想让这场比赛讨论的那个问题"。

[第二段：判准与讨论范围，80-120字]
判准必须前置。与定义自然衔接——"基于此，我方认为判断应不应该的标准是______"。讨论范围如有必要在此一并限定。

[第三段起：论点展开，每个论点 150-250 字]
论点开头用**"第一""第二"直标**，让评委第一秒就抓到你在论证什么。每个论点以**一段完整流畅的口播**呈现，段落内自然包含三层：①主张（这一段的中心判断）②说理（因果推演/对比论证/归谬反证）③证据（引用、案例、数据）。三者交融在行文中，不机械分段。

[末段：价值收束，60-100字]
不回扣论点编号做机械总结。把论证抬高一个层次，让听众感受到"这个持方意味着什么"。做实，不强行金句收束。
谢谢。
```

**格式硬要求**：

| 要求 | 说明 |
|------|------|
| **定义先行** | 开头即框定辩题核心词的含义，用理论/思想家/语境带入，忌照搬字典 |
| **判准前置** | 判准在论点之前明确写出，不可埋在行文中 |
| **一级论点 = 2 个** | 两个论点，每个以完整段落呈现，说理与证据交融 |
| **引用自然嵌入行文** | "黑格尔说……""正如……"——学术引用融入口语节奏，忌括号硬塞 |
| **论点开头直标"第一""第二"** | 论点段首句即亮出"第一/第二，……"，不埋在段落中间，评委一耳朵就能抓到 |
| **论点段落三层一体** | 每个论点段落内：主张（判断）+ 说理（推理）+ 举证（例子/数据），三者不机械分段 |
| **结尾做实不收束** | 不回扣论点编号做机械总结，不做强行金句——扎实收束论证即可 |
| **小明的正确用法** | 小明是内部框架工具，在破题阶段用于锚定立场与共情方向。一辩稿和二辩申论稿中**不出现「小明」这个名称**，但论证始终围绕小明所代表的那群真实的人——只是他们不叫小明。到了三辩小结和结辩，可以把「我们代表了谁」「底线是什么」讲出来——让评委感受到你是在为具体的人发声，而不是在玩文字游戏。本质上全程都有小明，只是前中期不点名，后期具象化 |
| **字数** | 全文 900-1100 字（约3分钟口播） |

**参考范例**（节选自第十届新国辩南大正一申论，辩题"当代年轻人应该活得更现实/更理想"）：

> 谢谢主席，各位好。今天我们要讨论的问题是理想与现实之争。也许在大家的认知中，理想积极而崇高，现实则消极、平淡乃至无意义，但这实属我们的误会。黑格尔说，现实比一切抽象的观念都更为深刻，现实即是一切事实的总和，它既蕴含着历史的沉淀，又潜藏着未来的潜能。理想则是某种观念或意识形态的外化。小时候，我们常说我的理想是成为科学家——哪怕那时候我们根本不知道科学是什么，科学家如何生活实践。但无需任何现实，理想依然成立，因为理想的实质就是某种意识形态所规划出的秩序等级：哪些是好，哪些不好，哪些先进，哪些落后。
>
> 而活得现实则是一切从现实出发。它在历史中的名称叫做实事求是。不是无意识地臣服于某些意识形态，而是试图解除这些遮蔽，将现实作为开发自己生活的原则。原来我们未必要30岁之前有车有房有存款，并且为此泣血拼搏；也可以认真而严肃地叩问自己的内心，从自身的感受出发，挖掘属于自己的答案。把自己作为方法，我们才可以解除理想的遮蔽。
>
> 最后，关于当代——这是一个东方与西方、新潮与旧派，各类价值观念与意识形态无孔不入又疯狂交织的混乱年代。未经审视的崇高背后是枷锁，不经反思的远方背后是谎言。当代年轻人的痛苦和无奈，往往来源于对理性权威的服从。理想不该成为定义人们生活的方式，年轻人更应该从现实中体悟反思，寻找属于自己的道路。这方是一种自我掌控的人生。

> **分析**：全稿约650字（1分42秒的片段），没有"首先/其次/最后"的显式分隔，没有括号引注，黑格尔直接入行文。定义、论点、价值三者以一段流水式的口播连贯推进——这就是高水准一辩稿的质地。

### 6.3 模块②：四质一（质询对方一辩 + 拆论 + 接质防守）

> 四质一是全场第一个交锋环节——双方一辩申论完毕后，我方四辩质询对方一辩。与此同时，我方一辩刚刚被对方四辩质询完毕。因此备赛时这个模块需要同时准备进攻和防守。

**三件事**：

| 任务 | 内容 | 说明 |
|------|------|------|
| **① 质询对方一辩** | 针对对方一辩的立论框架，设计追问链 | 趁对方框架还新鲜、队友还没保护——直接撕开口子 |
| **② 拆解对方一辩框架** | 预判对方一辩怎么立论，逐点拆 | 拆判准 + 拆论点——判准拆了论点自然悬空 |
| **③ 我方一辩接质防守** | 预判对方四辩会怎么质询我方一辩 | 守住我方定义+判准，不给对方第一轮就撕开口子 |

---

**结构模板**：

```
[第一部分：质询对方一辩]

第1组：[战场名称]（针对[对方某论点/某前提]）

1. [确认问题]（确认对方前提/定义/事实）
2. [追问1]（逼出逻辑漏洞）
3. [追问2]（收网——确认矛盾或荒谬结论）
4. [（可选）兜底问题]

【设计意图】：[这组问题瞄准的击穿点]
【对方可能恶逃方式与应对】：[归谬型/派别型/弃攻型/自洽型] → [我方应对]

（按战场分组：定义 1组 + 判准 1组 + 每个对方论点 1组 + 价值 1组 + 兜底 1组。每组 3-4 题，每题 ≤ 15 字。）

[第二部分：拆解对方一辩框架]

【对方可能论点X：[对方论点一句话概述]】

对方论证逻辑：[对方会怎么论证这个点，2-3句概括其推理链]

我方拆解思路：
1. [拆解点1]：[拆什么——定义/逻辑/论据/价值？怎么拆？]
2. [拆解点2]：[同上]

定列比结话术：
- 定：[明确反驳对象]
- 列：[一句话复述对方] vs [一句话亮出我方]
- 比：[为什么我更好]
- 结：[这与持方的关系]

（预判覆盖：对方核心论点×2 + 对方价值攻击 + 对方替代方案，至少 4 项。）

[第三部分：我方一辩接质防守]

【质询预测X：[对方可能质询方向]】

对方质询：[模拟对方四辩连续追问，2-3轮]
→ 我方回应：[回答 + 用接质六项技巧标注（【让步】【升维】【举例】【识别】）]

核心防守点：[这个质询在撬动我方哪个论证要素？]

（预判覆盖：攻定义/攻判准/攻论据/攻逻辑链/攻价值，至少 5 个方向。）
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| **质询问题短** | 每题 ≤ 15 字，质询问题必须短，让对方答，你不替他答 |
| **每组有设计意图** | 让使用者知道这组问题的「击穿点」在哪 |
| **每组有恶逃应对** | 预判对方四类恶逃（归谬/派别/弃攻/自洽），提前备好话术 |
| **拆论必须带定列比结** | 不能用「我觉得他说得不对」收尾 |
| **拆论先复述再拆** | 忌「还没听懂就开打」——先确认理解正确 |
| **接质预判 ≥ 5 个方向** | 定义/判准/论据/逻辑/价值全部覆盖 |
| **接质回应用技巧标注** | 【让步】【升维】【举例】【识别】——具体引用自 §2.2.2 |

**四质一 vs 三质二的区别**：

| | 四质一 | 三质二 |
|------|--------|--------|
| **质询对象** | 对方一辩（立论框架） | 对方二辩（申论内容——含对方对我方的反驳） |
| **对方状态** | 刚从一辩申论下来，还没有被队友保护过 | 已经过一轮攻防，论点已加固、漏洞已修补 |
| **我方策略** | 趁热打铁——在对方框架最裸露的时候打 | 打补丁——对方二辩加固了什么就拆什么 |
| **拆论对象** | 对方一辩的立论框架（预判） | 对方二辩的申论内容（含反驳己方+延伸论点） |
| **接质防守对象** | 我方一辩——重点守定义+判准 | 我方二辩——重点守延伸论点和己方反驳成果 |

### 6.4 模块③：二辩申论稿格式

> 二辩申论是全场**唯一兼具「驳」与「立」双重任务的中盘环节**。它既要拆掉对方框架，又要为之后的攻防铺路——比一辩稿更有锋芒，比纯反驳更有建设性。

**两件事**：

| 任务 | 内容 | 占时长 |
|------|------|--------|
| **① 反驳对方框架** | 打判准，和/或拆对方核心论点 | ≈1分钟 |
| **② 延伸己方一个论点** | 选己方一辩的一个论点做深做透 | ≈1分钟 |

**顺序不固定**：攻防形势决定先驳后立还是先立后驳。对方立论有明显漏洞可以先反击抢占主动；己方框架需要先说清楚可以先延伸再顺势回击。按实际需要调整。

---

**结构模板**：

```
[第一部分：反驳对方框架]

打判准（选打时用）：
直击对方判准的合理性/可辩性/启发性问题。不纠缠定义细节，一个问题一个问题往下追：
"对方辩友，你方说判断标准是XXX——但这个标准真的能帮我们判断这道题吗？如果一个标准让辩题失去了讨论空间/违背了常识/对讨论没有启发，那这个标准本身就需要被质疑。"

拆论点（选打时用）：
挑对方最核心的一个论点，用定列比结骨架快速拆解：
- 定：对方说[一句话复述]
- 列：但这个论证的问题在于[指出漏洞——定义偏了/逻辑断了/论据不成立/价值不对]
- 比：你方说的是XXX，而实际的情况是XXX
- 结：所以这个点不成立/不支撑对方持方

如果判准和论点都打，先判准后论点——判准拆了，论点自然悬空。

[第二部分：延伸己方论点]

选我方一辩的一个论点，把一辩没时间展开的内容做透：
"回到我方立场。一辩刚才提到了[某论点]，因为时间关系没能展开——这里我把它讲清楚。"

深的方向（选一个方向做透，不贪多）：
- 算清楚：把一辩定性描述的判断，用具体的成本/数据/逻辑链量化。"这个选择的成本到底是什么？我给大家算一笔账……"
- 补前置：解释一辩用了但没来得及定义的概念，铺垫后续攻防需要的共识。
- 换角度：用更通俗的语言、更贴近生活的例子，让一辩的抽象论证变得可感。
- 预堵反拆：正面回应"对方接下来可能会说XXX"，在对方开口之前堵住这条路。

最后用一句话把延伸的论证回扣判准或持方意义。

谢谢。
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| **反驳只打1-2个点** | 两分钟装不下三个拆解，挑最大的打 |
| **延伸只选己方1个论点** | 把一个论点做透，远好于两个论点各摸一下 |
| **延伸必须有实质增量** | 不能只是换了个说法——要有数据/算账/新角度/新例子 |
| **不重复一辩** | 延伸≠复读。一辩说过的结论一句带过，重点在「为什么」和「到底是什么样」 |
| **反驳有定列比结骨架** | 不能「我觉得不对」就过——说清对方哪里不对、为什么、实际是什么 |
| **字数** | 600-800字（约2分钟口播） |
| **顺序灵活** | 先驳后立还是先立后驳取决于场上形势，模板只是列出两部分，不规定先后 |

**延伸方向选择指南**：

| 什么情况下 | 选什么方向 |
|-----------|-----------|
| 一辩的论点是定性判断，缺乏说服力 | **算清楚**：把判断变成具体的数字/逻辑推演 |
| 一辩用了概念但没解释，后续攻防需要铺垫 | **补前置**：把概念定义清楚，给队友铺路 |
| 一辩的论证偏抽象，评委可能没听懂 | **换角度**：用生活中的例子重新讲一遍 |
| 预判到对方主攻方向，想提前设防 | **预堵反拆**：把对方的攻击路径封死 |

**参考范例**（辩题：年轻人应该回到大城市还是小城市，正方·大城市）：

> 对方辩友刚才说，判断标准是"哪种选择更能让年轻人获得幸福"。但我必须指出——如果这个标准只看结果不看成本，那任何选择都可以说"以后会幸福的"。一个刚毕业的年轻人，在北京租房押一付三要两万块，送外卖要跑三个月才能攒出来。这三个月，他在城中村隔断间里，每天通勤三个小时，下雨天还在给电瓶车披雨衣。对方辩友说"熬过去就幸福了"——但请问，在熬的每一天、每一个月，他到底是活得幸福，还是在用"未来会幸福"的念头说服自己眼下可以忍受？判断标准不能只看终点不看路径——如果通往幸福的路上全是消耗，那这条路本身就在吃掉幸福。
>
> 回到我方立场。一辩刚才讲了大城市的机会和资源，时间有限只提了一句"成本被低估了"——这里我把它讲清楚。大城市的机会不是免费的。你在小城市月薪五千，房租五百，每个月还有三千块可以学技能、社交、储蓄、生活。大城市月薪一万，房租四千，交通通讯两千，吃饭两千——到手两千，不如在小城市。而且更关键的是：大城市用高成本置换掉了你最值钱的东西——时间。你的25岁到30岁，每天四个小时在地铁上。这四年你可以拿来考一个证、学一门手艺、或者单纯地好好生活——但在大城市，这四年已经提前卖给了通勤。如果老家的机会是用你的一小时换来一点成长，大城市的"机会"是用你的两小时换同样的成长——那这到底是机会还是消耗？这道题的账，不能不算。

### 6.5 模块④：三质二（质询对方二辩 + 拆论 + 接质防守）

> 三质二发生在双方二辩申论之后——我方三辩质询对方二辩。此时对方二辩已经完成了一轮申论（加固了己方框架、反驳了我方立论、延伸了一个论点），因此三质二的进攻方向与四质一不同：**不打赤裸的立论框架，打对方加固和反驳的部分**。

**三件事**：

| 任务 | 内容 | 说明 |
|------|------|------|
| **① 质询对方二辩** | 针对对方二辩申论中的加固、反驳和延伸，设计追问链 | 打他刚刚说的话——他说了什么就打什么，不是重复打一辩 |
| **② 拆解对方二辩申论** | 预判对方二辩会怎么加固框架、怎么反驳我方 | 拆他对己方框架的加固 + 拆他对我方的反驳 |
| **③ 我方二辩接质防守** | 预判对方三辩会怎么质询我方二辩 | 守我方申论延伸的论点 + 守反驳成果 |

---

**结构模板**：

```
[第一部分：质询对方二辩]

第1组：[针对对方二辩对某论点的加固]（他加固了什么就打什么）

1. [确认问题]（确认对方二辩刚才的加固——"你刚才说XXX，意思是YYY对吧？"）
2. [追问1]（逼出加固中的薄弱环节）
3. [追问2]（收网）

【设计意图】：[这组问题瞄准的击穿点——他加固的哪个位置最脆弱]
【对方可能恶逃方式与应对】：[归谬型/派别型/弃攻型/自洽型] → [我方应对]

第2组：[针对对方二辩对我方的反驳]（他反驳了我方什么就打回去）

1. [确认问题]（"你刚才反驳我方XX论点时的依据是YYY，对吗？"）
2. [追问1]（指出他反驳中的漏洞）
3. [追问2]（收网——他的反驳不成立）

【设计意图】：[打掉对方对我方的攻击，回收我方论点]
【对方可能恶逃方式与应对】：…

（3-4 组，覆盖：攻他加固的论点 + 攻他延伸的内容 + 攻他对己方的反驳 + 兜底）

[第二部分：拆解对方二辩申论]

【对方二辩可能论点X：[对方会怎么加固某论点/怎么反驳我方某点]】

对方申论路径：[他大概会说——先加固XX论点（用XXX角度/新证据），再反驳我方YY论点（用XXX逻辑）]

我方拆解思路：
1. [拆解点1]：[拆什么——他加固的哪部分不成立？]
2. [拆解点2]：[他反驳我方的哪部分站不住脚？]

定列比结话术：
- 定：[明确反驳对象]
- 列：[他说的] vs [实际的情况]
- 比：[为什么他的加固/反驳不成立]
- 结：[这与持方的关系]

（预判覆盖：对方加固核心论点×1-2 + 对方延伸论点 + 对方对我方的反驳 + 对方替代方案）

[第三部分：我方二辩接质防守]

【质询预测X：[对方三辩可能质询方向]】

对方质询：[模拟对方三辩追问，2-3轮——会重点攻击我方二辩延伸的论点和反驳]
→ 我方回应：[回答 + 接质技巧标注]

核心防守点：[这个质询在撬动我方哪个论证要素？]

（预判覆盖：攻我方延伸论点的论据/逻辑 + 攻我方反驳的有效性 + 攻我方框架新暴露的薄弱点）
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| **质询针对二辩申论内容** | 不打已经过去的一辩——打他刚刚说的话 |
| **优先攻击对方的反驳** | 对方二辩攻击了我方什么，三质二的首要任务就是打回去——不让我方论点死在前半场 |
| **拆论区分「加固」和「反驳」** | 对方二辩申论有两层：加固己方和反驳我方。拆论时两层都要覆盖 |
| **接质重点守延伸论点** | 我方二辩刚延伸完的论点最脆弱——对方三辩会优先打这里 |
| **其余要求同四质一** | 质询问题 ≤ 15字、每组有设计意图和恶逃应对、拆论带定列比结、接质用技巧标注 |

**与四质一的区别**（复用对比）：

| | 四质一 | 三质二 |
|------|--------|--------|
| **质询对象** | 对方一辩（立论框架） | 对方二辩（申论内容——含加固+反驳己方+延伸） |
| **对方状态** | 刚从一辩申论下来，框架还裸露 | 已经过一轮攻防，论点已加固、漏洞可能已修补 |
| **我方主攻方向** | 打立论框架——判准和论点 | 打加固和反驳——他补了什么我拆什么，他打了我什么我打回去 |
| **接质防守对象** | 我方一辩——守定义+判准 | 我方二辩——守延伸论点和己方反驳成果 |

### 6.6 模块⑤：三辩小结格式

> 三辩小结是中场的最后一次发言。打到这个环节，比赛走势已经基本明朗——前面的质询和申论决定了局势，小结和上一轮质询一起完成了攻防收束。之后的自由辩只是收割，小结就是把「这场比赛到底怎么赢的」讲清楚。

**四件事**：

| 任务 | 内容 | 占时长 |
|------|------|--------|
| **① 收质询** | 把上一轮质询没打完的问题继续讲透——补掉对方最后的退路 | ≈30秒 |
| **② 补框架** | 我方立论里一辩来不及展开、二辩只延伸了一点的——还有没说清楚的，现在补上 | ≈30-40秒 |
| **③ 做比较** | 我方框架 vs 对方框架，比干净——哪套解释力更强、更不自相矛盾 | ≈20-30秒 |
| **④ 给决胜点** | 一句话收束：这场比赛最关键的分歧是什么，为什么我方赢了它 | ≈10-15秒 |

---

**结构模板**：

```
[第一部分：收质询]

接上一轮质询，把对方最后一条退路封死：
"刚才质询中，对方辩友对[关键问题]的回答是[对方的回应]。但这个回答的问题是——[指出对方回应中的漏洞/不成立的地方]。如果[对方的说法]成立，那[推导出对方无法接受的结论]。所以对方其实没有回答这个问题——[把问题重新摆清楚]。"

注意：不是重新质询一遍，而是作为旁观者把刚才那轮交锋讲透。质询是你和对方对话，小结是你对着评委说「刚才那轮发生了什么、为什么我们赢了」。

[第二部分：补框架]

我方框架里还有没讲清楚的地方——现在补上：
"回到我方立场。我方今天说[一句话重复我方核心主张]。对于[某个具体问题]，我方一辩提到了但时间有限没有展开——这里我继续往下讲。[补充论证——算清楚/补前置/换角度/延伸逻辑链]。"

补的内容必须和一辩+二辩形成递进关系，不是在重复。一辩给框架，二辩延伸了一个论点，小结给的是「这件事再往下挖一层是什么」。

[第三部分：做比较]

把双方框架并排放，比干净：
"各位，今天双方各给了一套说法。对方面对的问题是[对方框架的致命缺陷——自相矛盾/解释不通/漏了某类案例]。而我方框架[一句话说清楚我方框架的优势——更自洽/覆盖面更广/更符合常识]。"

干净比较不是每个点重新打一遍，而是跳出具体的攻防回合，给评委一个整体判断——「听了这么多，两边到底差在哪」。

[第四部分：给决胜点]

一句话收束，给出你认为这场比赛最关键的分歧：
"所以今天这场比赛，说到底就是看一件事：[一句话概括核心分歧]。在这个问题上，[为什么我方已经论证完成，对方没有做到]。谢谢。"
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| **收质询不是重复质询** | 质询是「问对方」，收质询是「对着评委说刚才那轮质询的结果」。语气不同，功能不同 |
| **补框架必须有实质推进** | 不是把二辩申论换个说法再说一遍——是在二辩延伸的基础上再往下挖一层，或者补一个二辩没覆盖的角度 |
| **做比较是整体判断，不是逐个回锅** | 这轮小结不需要把每个论点重新打一遍。跳出回合，给评委一个宏观比较：「两套框架，哪套解释力更强」 |
| **决胜点一句话说清楚** | 不要铺垫，不要修饰——就是「这场比赛争的是X，我方赢了X，因为Y」 |
| **不回避劣势** | 己方确实有没回应好的点——可以认，可以放，但不要在评委面前假装赢了它 |
| **字数** | 500-700字（约1.5-2分钟口播） |

**补框架的方向**（和二辩申论延伸的区别）：

| | 二辩申论延伸 | 三辩小结补框架 |
|------|-----------|-------------|
| **位置** | 比赛前半段 | 攻防收束后 |
| **作用** | 把一辩的论点做透，给队友铺路 | 把整场还没讲清楚的事讲完，不给结辩留坑 |
| **深度** | 选一个论点往深挖 | 可以不限于某一个论点——把框架里还没摊开的摊开 |
| **语境** | 还没听完对方的全部攻击 | 已经知道对方打完什么牌了——可以针对性补漏 |

**参考范例**（辩题：年轻人应该回到大城市还是小城市，正方·大城市，正方三辩小结）：

> 刚才质询中，我问对方辩友一个问题：如果大城市的机会确实更多，但成本吃掉了大部分收益，那「机会更多」本身还构成选择的理由吗？对方辩友的回答是「年轻人愿意付出成本」。各位，这个回答漏掉了一个关键的东西——「愿意付出」不等于「付出是划算的」。每一个被高房租吃掉积蓄的年轻人，在被吃掉的那个月也是愿意的。但愿意，没有让他的账户多出一分钱。对方用一个态度问题绕掉了一个成本问题——这轮对话的实质是，他绕开了。
>
> 回到我方立场。我方今天说这道题不能只看机会的峰值，要看成本和收益的净结果。一辩讲了机会和资源，二辩给你们算了一笔大城市的生活账。我这里往下挖一层——不只是生活成本的账，还有心理成本的账。你在小城市，失败了还有人兜底，你爸妈在家，朋友在隔壁，你摔一跤有人扶。大城市呢？你加班到凌晨两点走出写字楼，一条街上只有你一个人。这种孤立不是「磨练」，是持续的消耗——它消耗的是你下一次再试的勇气。成本不只是钱，还有你反复受挫之后还敢不敢再来的那颗心。
>
> 两边比到这儿，其实已经很清楚了。对方给了你一个画面——大城市机会遍地，你只要够拼就能赢。但他从头到尾没有解释：那九千九百九十九个拼了也没赢的人，在这套叙事里放在哪里？我方的框架从不回避这些人——因为我们的账算的不只是少数赢家的收益，而是每个普通年轻人要付出什么、能得到什么。两套说法放在一起，哪套更诚实，哪套覆盖了更多人，各位心里有判断。
>
> 所以今天这场比赛，说到底就是一件事——评价「该不该回大城市」，是拿塔尖上的人当样本，还是拿普通人当样本。谢谢。

### 6.7 模块⑥：自由辩攻防要点

> 自由辩是四分钟的全员混战——没有固定发言顺序，没有预写稿子，没有彩排。所以这个模块不出「稿」，出的是**战场地图 + 攻防火力配置 + 临场话术弹药**。

**三部分**：

| 内容 | 说明 | 核心问题 |
|------|------|----------|
| **① 战场规划** | 按优先顺序列出要打的战场，每个战场配核心论点 + Blocking话术 | 「我们打什么、不打什么」 |
| **② 攻防节奏** | 4分钟怎么分——哪个战场先打、花多久、什么时候转 | 「什么时候该转战场了」 |
| **③ 临场应对** | 预判对方最可能打回来的3-4个点 + 简短回击话术 | 「对方打回来的时候怎么挡」 |

---

**结构模板**：

```
[战场规划]

战场一：[战场名称]（优先级：最高——这个战场赢了比赛就赢了）
· 核心论点：[我方在这个战场上的一句话主张]
· 推进话术：[我方主动出击时说什么，12-15秒口播体]
· 挡（Blocking）：对方打回来时 → "[简短回应，10字以内，堵住就转]"
· 转战场信号：[当出现什么迹象时转下一个战场]

战场二：[战场名称]（优先级：高）
· 核心论点：[一句话]
· 推进话术：[12-15秒]
· 挡（Blocking）：[10字以内]
· 转战场信号：[…]

战场三：[战场名称]（优先级：中——补充伤害，不纠缠）
· 核心论点：[一句话]
· 推进话术：[12-15秒]
· 挡（Blocking）：[10字以内]
· 转战场信号：[…]

（通常 3-4 个战场。优先顺序：先争定义/判准 → 再争核心论点 → 最后收价值。）

[攻防节奏]

· 前1分钟：[先打哪个战场，争取什么效果]
· 第2分钟：[视情况转进哪个战场]
· 第3分钟：[巩固优势/追打对方回避的问题]
· 最后1分钟：[收束 + 把对方未回答的问题摆给评委]

每个战场耗时约1分钟，4-5个发言次。每次发言12-15秒。

[临场应对——对方最可能打回来的点]

对方打：[预判对方自由辩最可能推的论据/例子/追问]
→ 我方挡并转："[一句话挡掉 + 顺势转回我方战场]"

（预判 3-4 个方向，每个方向 ≤ 20字话术。自由辩没有时间展开——挡完立刻转。）
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| **战场不超过4个** | 4分钟自由辩装不下更多——该放的要放 |
| **每个战场有Blocking话术** | Blocking是自由辩的核心操作——堵住对方的口，让你继续推战场（§3.3） |
| **每句推进话术12-15秒** | 自由辩发言短，话术必须是能直接念的口播体，不能是提纲 |
| **Blocking话术≤10字** | 挡完就转，不纠缠——「你说的有道理，但核心是XXX」「这个例子恰恰说明我方……」 |
| **优先顺序明确** | 全队必须统一——先打哪个、什么信号转场、什么战场不追 |
| **转场信号具体** | 不能写「差不多就转」——写「当对方第三次回避成本问题时转战场二」 |
| **临场应对预判≥3个** | 对方一定会打回来的点——提前备好一句话挡回去 |

**与战术理论的关系**：本模块的操作依据来自 §3 自由辩论全部内容——战场起承转合（§3.2）、Blocking（§3.3）、互动关系（§3.1）、节奏控场（§3.5）。模板是把理论翻译成场上可执行的指令。

### 6.8 模块⑦：结辩稿格式

**结构模板**：

```
[第一段：争议梳理]（直面核心分歧，不要回避劣势）
"今天这场比赛，我们真正在争的是什么？我们争的是……对方辩友给出的方案是……但这其实是……"

[第二段：框架强化]（重申我方定义-判准-论点闭环，1-2段）
"各位请想一想……[回扣最核心的一个论据或案例]"
"今天我方的逻辑其实很简单：……[一句话收束全场比赛]"

[第三段：价值升华]（回到我方持方所代表的人群，情感打动）
"最后让我回到[破题时小明所代表的那些人]……"
"我们主张XX，不是在……而是在……[纠正对方贴的标签]"

[末句：做实收束，不强行金句]
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| 直面核心争议 | 对方追打什么，结辩就回应什么——不能假装没发生过 |
| 淡化逻辑/事实 | 结辩不是二辩，不要重新引数据、拆逻辑 |
| 回归价值与情感 | "幽默是最好的防守，感动是最好的进攻" |
| 不要重复队友 | 不做"我方二辩举证的XX数据对方没回应"的技术性归纳 |
| 纠正标签 | 如果对方整场在给我方贴标签（"你们太苛责""你们太冷漠"），结辩必须摘掉 |
| 结尾做实不收束 | 结辩同样不需要强行金句——扎实收束即可 |
| 字数 | 500-800字（约2-2.5分钟口播） |

### 6.9 模块⑧：资料卡格式

**表格模板**：

```
| 编号 | 适配论点 | 核心实证/案例 | 上场话术 | 对方可能的反拆 → 我方反拆 |
|------|----------|---------------|----------|---------------------------|
| 01 | 论点一：XXX | 【机理/数据/案例简述】… | "……"（口播体） | 对方可能说"……" → 我方回："……" |
| 02 | 论点二：XXX | 【机理/数据/案例简述】… | "……"（口播体） | 对方可能说"……" → 我方回："……" |
| … | 底线防守 | 【逻辑闭环/价值兜底】… | "……"（口播体） | 最坏情况：对方质疑"……" → 我方兜底："……" |
```

**每张卡的内容**：
- **核心实证/案例**：一句话——机理写推演链，案例写「谁+做了什么+为什么算我方立场」。忌长篇。
- **上场话术**：完整口播句，用引号包裹，拿起来就能念。
- **对方可能的反拆 → 我方反拆**：预判对方会怎么打这个例子，只给**最直接、一击封喉的一条**。

**硬要求**：

| 要求 | 说明 |
|------|------|
| 每张卡精简 | 每列一眼扫完，场上没时间翻长篇 |
| 话术口播体 | 直接可念，不要「可以说……」的提示语 |
| 反拆只给一条 | 不求穷尽，给最贴合、最直接的 |
| 至少每个论点1张 | 底线卡至少1张 |

**资料卡的使用方式**：资料卡不绑定单一环节——一辩稿引用时翻对应卡的话术、质询时翻反拆预判、自由辩时翻Blocking话术。它是整个订制包的论据基础设施，放在末尾供全队随时取用。

> 注：资料卡编排在订制包末尾而非开头，是因为备赛实际流程是先出稿、再配资料——而不是反过来。场上使用时也只需翻到需要的卡片，不需要顺序阅读。

[作者]. [标题].[J]. [期刊], [年份], [卷]([期]): [页码].

### 6.11 输出原则

**核心原则：按需给模块，不要默认全量输出。** 用户要什么就只给什么，不要擅自附带其他模块。

| 用户请求 | 输出 |
|----------|------|
| "写一辩稿" | **仅①**，不附带资料卡 |
| "准备四质一" | **仅②**（含质询对方一辩 + 拆论 + 接质防守） |
| "写二辩申论稿" | **仅③** |
| "准备三质二" | **仅④**（含质询对方二辩 + 拆论 + 接质防守） |
| "写三辩小结" | **仅⑤** |
| "准备自由辩" | **仅⑥** |
| "写结辩" | **仅⑦** |
| "要资料卡" | **仅⑧** |
| "一辩稿+资料卡" 或明确提到两个模块 | 用户点了几个给几个 |
| "全套/订制包" 或 "全部" | ①→⑧ 全部8个模块 |
| "破题分析" | 新剑宗四问框架 + 价值阶梯（头脑风暴，不进入订制包格式） |

当用户请求模糊、不确定需要哪个模块时，先问一句再产出。
