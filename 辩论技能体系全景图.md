# 辩论技能体系全景图

> 基于辩论筑基·1.0版全套课程体系 + 超辩学院专题课程整理
>
> **核心理念**：辩论的本质不是"做题式的逻辑比拼"，而是"有温度的价值沟通"——**为"小明"发声，在观众心里构建认同。**

---

## 〇、总纲：辩论的核心框架

### 0.1 辩论的本质

> **辩论的核心对象是人，目的是改变人。**

但人无法被别人直接改变——人只能被自己本就认同的事情重构平衡、自主选择改变。因此，辩论不是冷冰冰的逻辑模板，而是要在观众心里构建出一套"认识→取舍→确认"的逻辑，让听众通过共情和认同，主动站到我们这边。

**核心命题（新剑宗理论 — 李微中）**：辩论是立足事实，借用逻辑，指向价值的魔术。

```
事实  ←→  逻辑  ←→  价值
 ↑         ↑         ↑
论据      论证      持方意义
```

三者不可得兼，但好论者必在三者之间找到最佳平衡。

### 0.2 框架的三位一体：立场 + 方案 + 价值

任何一场高质量的辩论，框架都离不开三个核心模块，层层递进、相互支撑，且始终围绕"说服观众"展开：

**① 立场：明确"我们为谁而辩""主张什么"**

立场不只是简单的持方，而是要把辩题语言"翻译"成无歧义的"生活语言"，明确核心利益与理由，更要塑造一个具体的**"小明"**——他是你要为之发声的对象，是观众能共情的缩影。

> 示例：辩题"优绩主义是过时的教育理念"，我们的"小明"可以是"出身普通、资源匮乏，再努力也难突破阶层的高中生"，立场就可以翻译为"优绩主义'努力+天赋=资源分配'的逻辑，忽视了像小明这样的群体面临的资源不公"。

核心要求：态度明确一致，核心主张的论证完整，且所有观点都围绕"小明"的处境与诉求展开——让观众觉得你不是在空谈逻辑，而是在为真实的人发声。

**② 方案：提供"如何实现主张"的路径**

方案不是政策辩的专属，而是所有辩题打破僵局的关键。当双方都论证了自身的利弊，就需要通过方案强化利好、回避弊害，让"小明"的诉求有落地的可能。

方案可以是行为模式或政策倡导，无需复杂细节，但要贴合"小明"的实际。核心作用：让论证从"空谈定性"落地，让观众看到"我们的主张能真正帮到小明"。

**③ 价值：守住"主张背后的深层意义"，唤起共鸣**

价值是框架的"保底项"——即便事实战场僵持甚至处于劣势，仍能靠价值翻盘。它不是单纯煽情，而是通过连接"小明"的处境与更宏大的社会背景、理想期待，让观众认同我们主张的正当性。

> 示例：辩题"优绩主义是过时的教育理念"，价值可落脚"教育的核心是育人，不是筛选；当代社会更需要兼顾公平与人性的教育理念，让每个'小明'都有成长的机会"。

### 0.3 四大基础构成要素

核心模块需要具体要素支撑，这四个要素贯穿整场比赛，存在明确的逻辑关联：

```
定义（框定入口）
  ↓ 引导
讨论范围（限定边界）
  ↓ 限定方向
标准 / 零级论点（判断核心标尺）
  ↓ 拆分支撑
论点体系 / 一级论点（分维度论证）
```

**① 定义：框定讨论范围的"入口"**

定义的核心作用不是照搬字典，而是引导观众对辩题形成具体解读（即我方"小明"最关注的解读），进而框定讨论范围。

关键技巧：用语境带入定义，而非生硬罗列。可利用"人会直觉省略讨论范围"的原理。例如"解甲归田"辩题，反方可通过"战争未结束、个人有余力"的语境，自然框定讨论范围，而非单独给"解甲归田"下定义。

**② 讨论范围：限定"我们聊什么"**

讨论范围是定义的延伸，明确辩题的主体、场景、边界，避免跑题。它与定义、标准的关系：定义引导讨论范围；讨论范围限定标准的方向（同一讨论范围下可能有多个标准，但标准不能脱离讨论范围）。极端情况下，讨论范围可直接等价于标准。

**③ 标准 / 零级论点：判断持方成立的核心标尺**

标准是框架的"主心骨"，是判断我方立场成立的核心依据，本质是"零级论点"（最大、最上位的论点）。它不是一条简单的及格线，而是我方"小明"之所以坚定站在我方的最大理由，底层往往是"主体+价值"的结构。

- 单题制（如"中国应废除死刑"）：标准可围绕"需根解损"展开
- 双题制（如"人性本善/本恶"）：标准可设定为"哪种人性假设更能让小明这样的普通人实现良性成长"

标准的合理性需靠上位价值支撑。

**④ 论点体系 / 一级论点：支撑标准的分论点**

论点是对标准（零级论点）的拆分，用于分维度论证标准的合理性。每个一级论点需符合"论点句 + 双重媒介 + 总结"的结构，且始终围绕"小明"的诉求展开：

- **论点句**：精炼概括观点，无需论证，仅为铺垫
- **媒介1**：推理与理由（因果论证或对比论证）
- **媒介2**：数据、案例、学理等现实背书
- **总结**：串联理由与证据，说明"这些证据能证明小明的困境，也能支撑我方标准"

**论点拆分的三种方式**：

| 方式 | 结构 | 适用场景 |
|------|------|----------|
| **并集式** | 多个论点共同构成标准的全集 | 标准可拆分为并列维度 |
| **线性式** | 论点按逻辑链条递进 | 需因果链条推进的辩题 |
| **价值补强式** | 第一个论点支撑标准，第二个论点升华价值 | 需要兼顾逻辑与感动的辩题 |

### 0.4 框架的层级逻辑：价值阶梯

好的框架不仅有核心要素，还有清晰的层级逻辑——通过"价值阶梯"，可实现论点与标准的升降格。

**核心逻辑**：

```
上位抽象价值（不证自明的核心：公平、成长、幸福、尊严）
        ↑ 往上追"为什么重要"
标准 / 零级论点（判断持方成立的核心标尺）
        ↓ 往下拆"怎么实现"
下位具体事实 / 一级论点（论点句 + 双重媒介 + 总结）
```

- **往上追**：具体论点 → 标准 → 抽象价值（回答"这件事背后的深层意义是什么"）
- **往下拆**：抽象价值 → 标准 → 具体论点（回答"这件事具体能落地成什么"）
- **中间的"标准"**，是连接抽象价值和具体行为的核心枢纽

**阶梯搭建三步法**：

| 步骤 | 操作 | 方法 |
|------|------|------|
| **1. 定中间标准** | 先明确辩论的核心标准，作为阶梯的中间枢纽 | 问：我方持方成立的核心判断依据是什么？ |
| **2. 往上搭价值** | 追问"为什么这个标准重要"，直到触及不证自明的核心价值 | 标准 → 考上好高中 → 获得优质资源 → 实现个人潜能 → 上位价值：尊重每个学生的成长可能性 |
| **3. 往下拆事实** | 追问"这个标准怎么实现"，拆成3-4个具体的一级论点 | 上位价值 → 标准 → 具体事实1、2、3… |

**常见误区**：

| 误区 | 错误表现 | 纠正要点 |
|------|----------|----------|
| 层级无递进 | 下位=标准=上位，三者重复 | 每一级必须能回答"为什么"，不能重复 |
| 脱离"小明" | 价值空泛如"促进社会进步" | 阶梯必须围绕"小明"的处境与诉求 |
| 下位事实不具体 | "短视频能学东西"这种模糊表述 | 下位事实要具体到"行为、场景、对象" |

### 0.5 命题类型与框架适配

**单题制（非共存型论证，多为政策辩）**：

- 特征：仅一方能证成命题，逻辑终局是"政策应实行"或"不应实行"
- 正方框架构成（需根解损）：

| 要素 | 含义 |
|------|------|
| **需要性** | 现况有严重问题 |
| **根属性** | 问题成因与政策相关 |
| **解决力** | 政策能解决问题 |
| **损益比** | 政策利大于弊 |

- 反方挑战路径：①攻击合题性 ②维持现况 ③微调现况 ④提出相抗方案

**双题制（共存型论证，多为价值辩）**：

- 特征：双方命题可同时为真，核心争夺"定义、讨论范围与标准"
- 攻防关键：争夺定义与标准的采纳理由 → 在己方框架下完成"论点-论据"闭环
- 评委判断逻辑：先看双方是否提出完整框架 → 再看排除理由 → 最后检视关联性

**双题制快速诊断：和解空间测试**

在破题阶段，先问一个前置问题：**双方有没有"和解空间"？**

> 和解空间 = 双方核心主张可以同时成立、不存在必然冲突的区域。

如果存在和解空间，那么**这场比赛必然是定义战**。因为不存在实质分歧，胜负完全取决于"谁来定义核心词"——谁的定义被采纳，谁就赢了。

诊断流程：
1. 写出双方核心主张（各一句话）
2. 问：两者能同时为真吗？
3. 若能 → 这场比赛是定义战 → 胜负取决于定义与标准的争夺
4. 若定义战 → 谁的框架能同时实现双方主张，谁就拥有结构性优势

> 例：「当代年轻人应该活得更现实/更理想」——正方"实事求是的方法论"与反方"追求真和善"并不必然冲突。故这场比赛从头到尾是定义战。正方"方法论"战场能同时容纳双方主张，反方"目的"战场却无助于方法论选择——正方结构性占优。

## 一、破题与框架搭建（赛前 + 一辩）

> **核心任务**：搭建完整框架（定义+讨论范围+标准+论点体系+方案+价值），塑造"小明"形象。

### 1.1 破题：新剑宗立论四问（李微中）

在搭建框架之前，先用四问锁定方向：

| 步骤 | 问题 | 目的 |
|------|------|------|
| 1 | 这个题实际上是在问什么？ | 寻找合适的可证伪处——切点 |
| 2 | 我方需要论证什么？ | 负担举证义务——怎么切 |
| 3 | 我方怎样才能完成论证？ | 寻找论证的方式与标准 |
| 4 | 这样理解对人有何意义？ | 人应该选择怎么行动 |

### 1.2 论的定义与好论的条件

> **论，是让某一持方得以成立的充分理由。**

- 持方不是事实判断，而是价值判断——价值判断的关键，是要对人产生意义
- 不同的持方对应着不同的世界 → 不同持方下，人会有不同的行为
- **定理**：如果论不同，人的行为不会完全相同；论必须具有可证伪性

**好论的五个条件**：

1. 会让人的世界观有所变化，并为之作出改变
2. 在确保攻防优势的前提下，具有可证伪性
3. 明确自己所需要负担的论证义务
4. 以某种方式负担之，达到足以抵消怀疑的程度
5. 论证义务完成 = 可证伪处未被证伪 = 充分理由

### 1.3 塑造"小明"：从抽象辩题到具体的人

"小明"是你要为之发声的对象，是观众能共情的缩影。塑造要点：

- **具体化**：给"小明"一个明确的身份、处境、困境
- **典型化**：他要能代表辩题涉及的核心群体
- **共情化**：让观众能感知小明的痛苦、渴望与诉求

> 示例：辩题"短视频对青少年利大于弊"，小明可以是"喜欢刷短视频、缺乏自控力但充满好奇心的初中生"。

### 1.4 定义与讨论范围的确立

**定义的核心技巧**：
- 用语境带入定义，而非生硬罗列字典释义
- 定义要为后续的标准和论点服务——要能自然引导出我方的讨论方向
- 可利用"人会直觉省略讨论范围"的原理，不单独给定义，而是用语境框定

**讨论范围的确立**：
- 明确辩题的主体、场景、边界
- 排除极端案例（既有天赋异禀又完全不努力的人）
- 讨论范围与标准的关系：讨论范围限定标准方向

### 1.5 标准的确立与论证

标准是我方框架的"主心骨"，底层结构通常是"主体+价值"。

**确立标准的方法**：
1. 明确"小明"最关注什么
2. 将"小明"的关注点翻译为可判断的标尺
3. 用上位价值论证标准的合理性

**驳斥对方判准的三个角度**（张子龙）：

| 角度 | 含义 |
|------|------|
| **合理性** | 这个判准是否符合常识？ |
| **可辩性** | 这个判准是否让辩题失去了讨论空间？ |
| **启发性** | 这个判准是否能让讨论更有深度？ |

### 1.6 立论撰写：从常识中来，到常识中去（李微中）

**立论应当包含**：
- **完整的逻辑链**：诠释辩题想干嘛 → 如何讨论辩题（双方应论证什么）→ 我方如何论证 → 补价值
- **事实与例子**：情景化解释 → 复杂概念生动化 → 论点佐证强化 → 通俗化
- **主次分明**：关键处 > 复杂处 > 会被挑战处 > 其他地方

**立论的核心目标——一个让评委懂但不要让对手懂的立论**：

| 原则 | 技巧 |
|------|------|
| 用浅显的论点预示深刻的内涵 | 实际理由必须通俗易懂，但选择这些理由大有玄机 |
| 用公允的讨论隐藏攻防的野心 | 立论过程要公允得不能再公允，但讨论方向指向己方攻防优势 |
| 用核心的论述辐射整个世界 | 只讲明白一件事，讲得通透而周全；之后再看待整个世界的任何质疑，无不在此论述之下 |

### 1.7 定义战：从世界观到价值观的完整操作链

> 定义战不等于"下定义"。定义战是一场比赛中决定胜负的战略形态——定义一旦站稳，进可攻退可守；定义一旦被拆，整个立论随之坍塌。

**定义的双层结构**：

| 层次 | 核心问题 | 标准 | 辩论操作 |
|------|----------|------|----------|
| **世界观** | "能不能这样定义？" | **不矛盾**——前后一致，能解释所有案例 | 归谬法起手：用符合对方定义但极度违背直觉的例子，逼对方要么认怂改口，要么硬着头皮解释 |
| **价值观** | "为什么要这样定义？" | **不违和**——符合人们的道德直觉与生活常识 | 建立理论说明"这样定义之后，我们能看到什么更有意义的东西" |

**世界观层的攻防**：

- 攻击方：通过归谬让对方陷入两难——坚持定义=违背直觉；放弃定义=定义战认输
- 防守方：只要做到不自相矛盾（同一类型的案例用同一标准评价），世界观层就能守住
- 常见错误：攻击方只攻世界观（"你这个定义好奇怪"），从不追问价值观（"你为什么非要这么定义"）→ 即使归谬成功，也只是让对方难堪，没有真正拆掉

**价值观层的攻防**：

- 进攻方必须追问："就算你能这么定义，你为什么要这么定义？凭什么你的定义比常识更优？"
- 防守方必须回答：这个定义让我们看到了______（常识看不到的东西），而这个东西之所以重要，是因为______
- 正方常见的价值观论证策略：我的定义是一种"方法论"（怎么做事），对方的定义是一种"目的论"（做什么事），方法论比目的论更根本——因为好的方法论能帮你实现任何目的，而只知道目的却不知道方法，可能南辕北辙

**定义战中的角色分工**：

| 环节 | 任务 |
|------|------|
| 一辩申论 | 给出定义，建立"世界观"的解释框架（能这样定义），铺垫"价值观"（这样定义的意义） |
| 被质询时 | 守住"世界观"——面对归谬，态度坚定，不自相矛盾 |
| 质询对方时 | 归谬起手→逼出对方世界观的矛盾→追问价值观 |
| 自由辩 | 用我方定义反复解释对方的例子（"你举的这个，恰恰是我方立场的体现"），展示定义的解释力 |
| 结辩 | 总结两种定义代表的两种世界观，说明我方定义更优的价值观理由 |

**定义战的致命时间窗口**：

定义战**必须在对方用自己的定义开始收割之前打掉**。一旦对方用定义拆完了你方所有论点，再来攻击定义为时已晚——裁判已经用对方的框架理解完整场比赛了。

## 二、攻辩交锋（二、三辩）

> **核心任务**：攻防框架漏洞，强化己方核心要素，维护"小明"的合理性。

### 2.1 质询体系（张子龙）

#### 2.1.1 质询的本质

> "质询的特性"是"对方可以即时地回答问题"。
> 因此，质询的**唯一目的**，就是**确认共识**。

只要确认，拆不拆无所谓。
- 不需要去反驳——"你说的是错的"
- 反驳也是确认——"你没有办法回答我的问题对吧？"
- 确认也是反驳——"你的论点是这个没错吧？"

**确认框架**：

```
论点 → 前提 → 结论
  ↓       ↓       ↓
有无？  有无/强度？ 证成命题？
                   其他可能/内部矛盾？
```

#### 2.1.2 质询的黄金原则

**① 确认、确认、再确认**

核心追问路径：
1. 他的论点和前提是什么？他有证明吗？他的证明有说服力吗？
2. 他的前提真的可以推出结论吗？
3. 他的结论真的有论证命题吗？

**② 学会问"为什么"**

连续追问，直到逻辑链条彻底暴露：
> 为什么跳槽不利于成长？
> → 因为青年人太自我中心
> → 为什么太自我中心？
> → 因为想法和社会不一样
> → 为什么这样就是自我中心？
> → 这样才能归因为自己，归因自己才有利于成长
> → ……（循环追问）

#### 2.1.3 质询的节奏与分寸

| 技巧 | 说明 |
|------|------|
| **打断时机** | 表达足够精确时才可打断，宁可慢半拍不抢一瞬 |
| **见好就收** | "没有趁胜追击这种事情，你原本打算问什么，就问什么！" |
| **黄金时长** | 每套问题控制在30-45秒，预算30-45%时间用于控场和应对干扰 |

**追问纪律：对方没正面回答时，至少再问两次**

比赛中常见一种致命失误：质询方问到了关键问题，对方换了个说法搪塞过去（"计划本身是活得现实的表现"），质询方却没有追打、自己转移了话题。

识别信号：
- 对方回答了一句话，但**意思等于原问题中的某个词换了说法**
- 对方用"你方的定义是……"反推，而不是正面回答你的问题
- 对方答非所问，开始举例子

应对纪律：
1. 识别到对方没正面回答 → 立刻重复原问题（可换一个问法，但同一指向）
2. 对方再次回避 → 第三次追问，并可直接点出："你没有回答我的问题，我的问题是______"
3. 对方三次回避 → 记录在案，转给后续环节："对方辩友刚才三次没有正面回答______，请评委记录"

> 注意：追问不等于纠缠。如果对方给出的是一个实质回答（即便是对你方不利的回答），就应该接住而非机械重复。追问纪律只针对"换说法搪塞"的情况。

#### 2.1.4 四种"无法回答"的应对话术

| 对方回应类型 | 破解方法 |
|-------------|----------|
| 归谬型恶逃 | "好嘛，不然你觉得人们是怎么想的呢？" |
| 派别型恶逃 | "我怎么知道别人怎么想"→ 切换为"你方的主张是什么？" |
| 弃攻型恶逃 | 直接指出对方回避，记录在案 |
| 自洽型顽固 | "你说的有道理，但这不能证明你的持方" |

### 2.2 接质体系（柴子凡）

#### 2.2.1 接质的根本目标

**守住己方的论证体系**。完整的论证体系有三层：
1. **事实前提**：排除合理怀疑（注意变量改变——客观条件、制度环境变迁）
2. **价值底线**：足够的素材与空间（价值底线无法依靠事实逻辑证成）
3. **逻辑推理**：强度契合、形式有效（演绎推理 + 当然推理）

#### 2.2.2 接质六项实用技巧

| 技巧 | 要点 |
|------|------|
| ① 回答优先级 | 实质内容 > 形式标准 > 空话套话 |
| ② 省略/缩短/后置让步 | 防止"话还没讲完就被打断" |
| ③ 答"超"所问 | 主动推进讨论进程，承认底线并渲染价值 |
| ④ 用例子代替逻辑 | 抽象逻辑链用直观例子替代 |
| ⑤ 虚例语境补全 | 主动补全对方虚例中未说明的前提 |
| ⑥ 警惕诱导发问 | 识别隐藏的未经确认的前提 |

### 2.3 反驳体系

#### 2.3.1 论证三要素

```
论点 Claim   — 核心观点，须满足：明确性 · 可辩性 · 相关性
论据 Evidence — 支撑血肉，须满足：权威性 · 时效性 · 代表性
论证 Warrant — 逻辑桥梁：演绎 · 归纳 · 类比
```

#### 2.3.2 反驳的四个维度

| 维度 | 核心 | 方法 |
|------|------|------|
| **一：反驳定义** | 釜底抽薪，重划战场 | 拆解概念边界 → 重构有利定义框架 |
| **二：反驳逻辑** | 斩断链条，识别谬误 | 识别稻草人/滑坡/以偏概全/因果混淆 |
| **三：反驳论据** | 去伪存真，质疑支撑 | 质疑真实性 + 相关性 + 充分性 |
| **四：反驳价值** | 揭示偏颇，升维打击 | 剖析对方价值预设 → 提出更优普世价值 |

#### 2.3.3 反驳四层次（黄执中）

| 层次 | 名称 | 表现 | 评价 |
|------|------|------|------|
| 1 | 反驳观点 | 仅说"我不同意"，无实质论据 | 最低级，情绪宣泄 |
| 2 | 反驳逻辑 | 指出逻辑谬误与推理漏洞 | 合格，能有效削弱 |
| 3 | 反驳前提 | 攻击对方隐藏的未言明假设 | 高级，釜底抽薪 |
| 4 | 反驳价值 | 上升至价值层面，提出更优解 | 最高级，破立兼具 |

#### 2.3.4 常见逻辑谬误速查

| 谬误 | 描述 | 经典例句 |
|------|------|----------|
| **稻草人** | 歪曲对方观点再攻击 | "如果你不支持这政策，你就是不爱国" |
| **滑坡谬误** | 夸大行动的连锁后果 | "今天允许同性婚姻，明天人和动物结婚" |
| **以偏概全** | 个别案例推普遍结论 | "比尔·盖茨辍学成首富，所以高教无用" |
| **因果混淆** | 相关性误作因果 | "冰淇淋销量上升→溺水事故增多，所以冰淇淋导致溺水" |

#### 2.3.5 高级反驳战术

**归谬法**：假设对方观点为真 → 顺着逻辑推导 → 得出荒谬结论 → 反证原观点错误

**借力打力**：承认对方部分合理性 → 利用其逻辑前提 → 证明这恰恰支持己方观点

**顺水推舟**：不否定对方前提 → 顺着逻辑延伸 → 导出对方无法接受的结论

**反驳强度等级**：
- 最弱：质疑（"你是不是不对？你怎么证明？"）
- 最强：击穿（拆掉："你这样不对，因为……" / 吃掉："你对了也没用，因为……"）

#### 2.3.6 临场反驳四步法（谢宇·定列比结）

| 步骤 | 动作 | 含义 |
|------|------|------|
| **定** | 定点 | 明确反驳对象——你要打什么？ |
| **列** | 陈列 | 复述双方观点——你说的 vs 我说的 |
| **比** | 比较 | 分出高下优劣——为什么我更好？ |
| **结** | 联结 | 指出立场关联——这与我们的持方什么关系？ |

#### 2.3.7 反反驳标准化模板

> 对方反驳你方论证时，通常有三种攻击方向。每种都有标准化的回应套路。

**对方说"A未必有B"（你的方法论未必导致你主张的好结果）**

| 回应 | 话术模板 |
|------|----------|
| **"没有A会更糟"** | "你举的例子如果按照我方方法论去做都这样了，那不按我方方法论去做的结果只会更差。" |
| **"不是A有问题，是做A不够好"** | "这恰好说明他还没有真正做到我方主张的方法论——如果他做得更好，就不会是这个结果。" |

**对方说"没有A也有B"（不按你的方法论也能成功）**

| 回应 | 话术模板 |
|------|----------|
| **"这是幸存者偏差/运气"** | "不按我方方法论也能成功的案例，是运气好——更多的同类案例失败了，只是你方没有举出来。" |
| **"有A会更好"** | "不按我方方法论都能做到这个程度，如果按了，岂不是能做得更好？" |

**对方说"A会导致C这个坏处"（你的方法论会导致某个恶果）**

| 回应 | 话术模板 |
|------|----------|
| **"这是别人介入造成的，不是A造成的"** | "导致坏后果的不是我方主张——是他人介入的行为。你不能把别人的错算在我方的头上。" |
| **"对方替代方案导致C的风险更大"** | "每一种选择都有风险。问题是，我方的方法论导致C的风险，比对方方案导致C的风险更小。" |

> 以上模板的核心思路：不否认对方指出的现象，而是**重新归因**或**升维比较**。这些不是哲学武器，只是反驳中的因果归因训练——辩手需要在模辩中反复练习，直到成为条件反射。

## 三、自由辩论（全员）

> **核心任务**：深化框架分歧，巩固己方优势，强化"小明"的诉求。

### 3.1 战场的两种类型（卢卡斯）

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| **推进主导型** | 自闭推进己方观点，呈现优势，常使用歌词模式 | 己方立论占优时 |
| **攻防主导型** | 重点在交锋上击溃对手，不封闭，与对手交流性高 | 需要撕开对方防线时 |

**战场之间的"互动关系"——决定谁能同时吃下双方利益**

一场比赛往往不只一个战场。审局的关键是判断**战场之间有没有互动关系**：

> 互动关系 = 甲方战场成立会对乙方战场产生影响，反之则无。

| 互动类型 | 含义 | 策略价值 |
|----------|------|----------|
| **单向影响** | A战场成立有助于B战场（如"方法论"有助于"实现目的"），但B战场成立对A战场无帮助 | 拥有A战场的一方结构性占优——因为他可以同时实现双方的主张 |
| **双向无关** | 两个战场互不影响 | 谁在两个战场上都占优谁赢；各赢一个则平局，比损益比 |
| **互相冲突** | A战场成立意味着B战场不成立，反之亦然 | 定义战的典型形态——争夺定义就是争夺哪个战场被计入胜负 |

**拥有互动关系优势的一方，最安全的打法是**：稳住自己的定义；收割所有的利益；用自己的定义拆解对方的一切论点。

> 例：正方"方法论"战场成立后，可以直接说"你方的理想很好，所以要用我方实事求是的方法论来实现这个理想"——正方同时吃下双方利益。反方"目的"战场即便成立，也无助于告诉年轻人具体该怎么做事。

### 3.2 战场的"起承转合"结构

```
起（阐明战场重要性）
  → 承（补充更多资料，输出伤害）
    → 转（做比较——拿对方劣势比己方优势）
      → 合（为战场做总结）
```

### 3.3 自由辩操作核心：Blocking（挡）

> 不是反驳，而是堵住对方的口，让你可以继续推演战场。

具体技巧：提相抗、追加责任、区隔缩小、顺手转。

### 3.4 三种攻防一体高阶招法

| 招法 | 公式 | 例 |
|------|------|-----|
| **前提型战场** | "要讨论XXX，就必须先讨论YYY" | "要讨论控枪的好处，就必须先讨论控枪的过程" |
| **内涵型战场** | 重新定义核心概念的内涵 | "拥有枪支是美国文化，但有害的文化也要传承吗？" |
| **盖罪名** | 提高对方论证成本 | "对方辩友您现在要做的是……" |

### 3.5 自由辩节奏控场（陈劭康）

- 以**4个战场**为基本规划（4分钟自由辩）
- 每个战场耗时1分钟，4-5个发言次
- 每个发言次控制在**12-15秒**
- 优先顺序：先争定义与讨论范围 → 再争标准 → 最后争论点

**团队控场三要素**：
1. **整体流动意识**：赛前统一战场顺序和负责人
2. **全程沟通**：像电竞/篮球，全程口头沟通
3. **中间辩位是沟通桥梁**：上传下达

#### 3.5.1 临场指挥四能力：听说读写

场上必须至少有一人能同时做到以下四项——这不一定是四辩，也不一定是队长，而是"团队中审局最快的那个人"。

| 能力 | 含义 | 具体操作 |
|------|------|----------|
| **听** | 迅速听懂场上形势 | 对方这个反驳瞄准的是我方哪个位置？是打定义还是打逻辑？ |
| **说** | 整理比赛型发言 | 把审局结论转化为简短指令："守定义""转战场""用运气打回去" |
| **读** | 看懂队友的记录 | 四人必须共用一套场上记录符号/体系 |
| **写** | 记清比赛形势，让队友看懂 | 关键攻防节点画图记录，不能只靠脑子记 |

**场上记录体系（最低共识）**：

| 记录要素 | 格式 |
|----------|------|
| 对方核心论点 | 编号+一句话（如"①理想=原则判断"） |
| 我方已拆标志 | 打钩（✓已解决）或打叉（✗待处理） |
| 对方未回应的我方追问 | 星号+内容（如"*对方未回答'为什么理想者不能有计划'"） |
| 战场状态 | 简短标注（如"定义战：我方优势""目的战场：平"） |

#### 3.5.2 审局与战场制图

> 审局不是在结辩时才做——审局是贯穿全场的连续动作。每过一个攻防环节，场上形势就更新一次。

**制图格式**（每次攻防环节结束后更新）：

```
┌─────────────────────────┐
│ 战场一：定义（方法论）      │
│ 正方：活得现实=实事求是   │
│ 反方攻击：葛朗台/陶渊明    │
│ 状态：正方守住（←暂未追问价值观）│
├─────────────────────────┤
│ 战场二：目的（追求什么）    │
│ 反方：求真求善/自由人格    │
│ 正方攻击：用实事求是更能实现│
│ 状态：正方拆解成功         │
└─────────────────────────┘
```

**制图判断依据**：

| 战场状态 | 标志 |
|----------|------|
| **一方优势** | 该方定义/论据对方未有效回应，或每次回应都被反拆 |
| **平局/未定** | 双方均有有效攻防，未分出明显高下 |
| **潜在威胁** | 对方问到了关键但未追打，随时可能被后续环节重提 |

**审局结论直接影响战术决策**：
- 若定义战场我方占优 → 命令全队"一切回归定义"，用定义打对方一切举例
- 若定义战场受威胁 → 优先在下一个环节加固定义，暂缓推进新论点
- 若对方某核心论点仍未被我方触及 → 指定下一轮质询/自由辩专门处理

### 3.6 对辩节奏三种模式

| 模式 | 操作 | 时间效率 |
|------|------|----------|
| **先守后攻** | 先回答再提问，最常用 | 两步操作，耗时较多 |
| **借力打力** | 用对方词汇组装问题问回去 | 一步操作，高效高技 |
| **自由变化** | 忽长忽短，让对方无法习惯 | 出其不意 |

## 四、结辩升华（四辩）

> **核心任务**：梳理框架交锋，强化己方优势，升华"小明"背后的价值。

### 4.1 结辩的核心定位

结辩是整场比赛的"收口"，不是重新立论、不是继续攻防，而是：
1. **梳理分歧**：明确双方框架的核心差异及攻防结果
2. **强化框架**：重申定义、讨论范围、标准与论点的闭环，维护"小明"的合理性
3. **价值升华**：结合社会背景，说明"支持小明"的深层意义

### 4.2 结辩操作要点

**该做的**：
- 直面核心争议——对方追打什么就回应什么
- 淡化逻辑/事实，回归价值与情感——这是结辩最强大的武器
- 展现气场与诚意，说"其实今天的比赛很简单"
- 把具体争议拉升到"这对小明意味着什么"

**不该做的**：
- 不重复队友——不做"我方二辩举证的XX数据对方没回应"的技术性归纳
- 不装作若无其事——有明显劣势时不回避
- 尽量不做前置攻防——结辩还在处理定义和判准，基本翻不了

**各环节的时间窗口纪律**：

| 环节 | 最晚必须处理的事 | 过了这个环节没处理=基本来不及 |
|------|-----------------|-------------------------------|
| **反一被质询后** | 识破正方是否在打定义战 | 若到自由辩才意识到，时间不够 |
| **反二申论** | 攻击正方定义（若正方靠定义吃饭） | 正方用定义拆完你方论点后再打定义，评委已用正方框架理解全场比赛 |
| **正三/反三小结** | 锁定胜负手，放弃已死战场 | 还在同时打三个战场=一个都赢不了 |
| **自由辩前半段** | 巩固前三个环节建立的框架优势 | 自由辩后半段只能收割，翻框架需要时间 |
| **结辩** | 放弃一切未解决的纯逻辑攻防，只做价值收束 | 结辩还打定义战=本场框架战已经输了 |

### 4.3 幽默与感动

> "幽默是最好的防守，感动是最好的进攻。"

结辩的语言要能直击人心——不是华丽辞藻的堆砌，而是回到"小明"的故事，让评委和观众在最后时刻感受到：我们不是在打一场逻辑游戏，我们是在为一个真实的人的处境而战。

## 五、专项能力

### 5.1 价值渲染（梁秋阳）

#### 5.1.1 核心公式

> **细节 → 真实 → 冲突 → 认同**

辩论的深层竞争是心智占领：
```
事实资讯 → 逻辑推论 → 观点共鸣 = 胜利
```

#### 5.1.2 价值渲染的三种进路

| 进路 | 方法 | 关键要点 |
|------|------|----------|
| **戏剧冲突** | 找到"相反与相对"，制造价值张力 | 公平的反义词不只是"不公平"，还可能是"效率" |
| **细节** | 把抽象描述变为具体枚举 | "他带着一只猴子、一只猪、还有一个河里的妖怪，走上十几年" |
| **真实（Keep it real）** | 直接 > 间接，私人心声 > 公共文本，断言 > 推论 | 用你自己的话，说真实的感觉 |

#### 5.1.3 价值冲突制造法

**找到价值的"反面"**：
> 正义 → 不公（相反） → 不义（矛盾） → 专制（负面之负面） → 稳定

每深入一层，价值的张力就增强一分。

**低打与高打**：
- **低打**："为什么"问句引导 → 曲径通幽
- **高打**："凭什么"犁庭扫穴 → 酣畅淋漓

#### 5.1.4 价值渲染实战模板（以"自由"为例）

| 版本 | 文案 | 效果 |
|------|------|------|
| 初始 | "自由是人类的天性，我们渴望自由" | 虚空，无共鸣 |
| +冲突 | "即使有100%安全的保证，代价是从此没有隐私，可以吗？" | 制造两难，引发思考 |
| +细节 | "代价是你每一秒都活在监察和控制之下" | 画面感增强 |
| +真实 | "我问你：如果保证你和家人再也不会受到伤害，代价是你必须每一秒都活在监察和控制之下，你会接受吗？……还是说，你也知道，在我们每个人心底里都还有那一分对自由的向往。" | 直击心灵 |

### 5.2 语境塑造（张子龙）

#### 5.2.1 语境塑造三层次

| 层次 | 方法 | 效果 |
|------|------|------|
| **1 增加细节** | 强化旧的语境 | 让原有框架更鲜活有力 |
| **2 改变脉络** | 建立新的语境 | 在具体与抽象之间切换 |
| **3 转换框架** | 塑造新的认知 | 给观众一副新的"眼镜" |

#### 5.2.2 核心工具：上堆下切法（NLP）

**上堆（抽象化）**：把具体问题升维
> 创业失败 → 学业和爱情失败 → "失败"本身 → 人生挫折

**下切（具体化）**：把抽象概念降维
> 财富 → 资产 → 农产 → 家畜 → 母牛 → 阿花（那只具体的牛）

#### 5.2.3 辩题框架重构三步

**步骤一：重设战场**（"这个题目有趣的部分不是……而是……"）
> 例：爱上人工智能算不算爱情 → 这个题有趣的部分不是爱情，而是当科技越来越进步，虚拟和现实还有没有差别？

**步骤二：铺陈战场**（用一个故事/电影/现象展开新视角）
> 例：《黑客帝国》Matrix，红药丸与蓝药丸的选择

**步骤三：价值抬高**（从具体情境升维到人类根本命题）
> "因为我们还在意真假，不是因为真实比较幸福，不然，没尊严！"

### 5.3 资料运用（唐梦）

#### 5.3.1 备赛资料四分类

| 类型 | 内容 | 用途阶段 |
|------|------|----------|
| **背景资料** | 题解描述、辩题来源、题设背景 | 解辩题 |
| **参考范例** | 参考辩案、论点、辩词、视频 | 头脑风暴 |
| **定义** | 专业术语定义、一般名词定义、合题案例 | 立论建构 |
| **论据** | 统计数据、调查报告、论文结论、专家证言 | 立论建构 |

#### 5.3.2 资料检索渠道

| 渠道 | 具体来源 |
|------|----------|
| 政府统计 | 统计公报、统计数据 |
| 商业报告 | 招股说明书、行业协会报告、券商行研报告 |
| 数据库 | 知网、万方、维普、北大法宝、WestLaw、国泰安 |
| 一手资料 | 电话访谈（政府/行业协会/企业）、问卷调查 |

#### 5.3.3 不同类型辩题的资料侧重

| 辩题类型 | 定义依赖度 | 论据依赖度 | 资料重点 |
|----------|-----------|-----------|----------|
| **政策类** | 低 | 高 | 数据、研究结论、专家证言 |
| **时事/商业类** | 较低 | 较高 | 学术理论（社会学/心理学/经济学/管理学） |
| **价值思辨类** | 高 | 低 | 合题案例的举例与阐释 |
| **脑洞类** | 较高 | 较低 | 辩题假设在现实中的逻辑推论 |

#### 5.3.4 资料拆解三板斧

| 方法 | 操作 | 杀伤力 |
|------|------|--------|
| **可靠性拆解** | 质疑样本容量、时间、方法、利益相关 | 适用于几乎一切论据，但只能削弱不能排除 |
| **关联性拆解** | 逻辑链拆解、类推归谬 | 杀伤力最强，可彻底破坏论据效力 |

#### 5.3.5 比赛中呈现资料的技巧

- **节选**：不应通篇念读，择选重点
- **修辞**：不应照搬原文，加以修辞
- **解读**：呈现资料后需注重解读
- **自由辩3秒原则**：资料呈现以3秒（约20字）为限；不问对方"知不知道"，直接提数据诘问

#### 5.3.6 举例前的硬核核实义务

> 一场比赛中，最致命的打脸不是逻辑被拆，而是**你举的例子恰恰是对方立场的例证**。当你用新闻事件、历史案例来支撑论点时，必须过一手信源，而不是依赖记忆或标题。

**核实清单（每个例子出稿前必答）**：

| 核实项 | 操作 |
|--------|------|
| **事实时间线** | 事件发生的准确时间、顺序——"赛后拒唱国歌"和"赛前拒唱"是完全不同的事 |
| **当事人一手说法** | 当事人自己怎么说的？有没有采访/手记/自述？优先一手信源，忌转述 |
| **因果方向** | 你自己认定的"因为理想所以做了X"——当事人真的是出于这个原因吗？还是有更复杂的综合考量？ |
| **结果确认** | 结果是成功了还是失败了？有没有被后续事件反转？ |

**常见信源层级**：

| 层级 | 信源 | 可信度 |
|------|------|--------|
| 一手 | 当事人采访/手记/自述、原始文件、官方公告 | 最高 |
| 二手 | 正规媒体报道（路透/新华社/NYT等）、学术论文 | 较高，但需交叉验证 |
| 三手 | 公众号/自媒体/百科/网友总结 | 仅作线索，不可直接当作场上论据 |

**致命红线**：
- **不要只看标题和简介就引用新闻事件**。读完全文，确认因果方向。
- **不要说"赛后"而不核实**。时间顺序错=当场被对方用事实打脸。
- **不要把复杂的政治/社会事件简化为"某人出于XX理想做了XX"**。真实世界中的决策远比动机单一的叙事复杂。

> 例：「伊朗男足拒唱国歌」——反方用此例支持"活得理想"：球员出于理想拒唱国歌。但一手信源显示：①这是赛前而非赛后；②球队2:6惨败；③球员采访称"从未过分关注此事"；④下一场比赛在当局威胁家人后立即恢复唱国歌。这个例子更接近"多方势力博弈下的现实权衡"，恰恰是正方立场的例证。

### 6.2 模块①：参考一辩稿格式

**结构模板**：

```
谢谢主席，各位好。

[第一段：定义框定与语境建立，100-150字]
从辩题核心词切入，建立一个能让听众重新理解辩题的框架。可以援引思想家/理论作为定义支撑（如"黑格尔说，现实即是一切事实的总和"），但不要生硬罗列字典释义。关键在于把辩题翻译成"我方想让这场比赛讨论的那个问题"。

[第二段：判准与讨论范围，80-120字]
判准必须前置。与定义自然衔接——"基于此，我方认为判断应不应该的标准是______"。讨论范围如有必要在此一并限定。

[第三段起：论点展开，每个论点 150-250 字]
论点开头用**"第一""第二"直标**，让评委第一秒就抓到你在论证什么。每个论点以**一段完整流畅的口播**呈现，段落内自然包含三层：①主张（这一段的中心判断）②说理（因果推演/对比论证/归谬反证）③证据（引用、案例、数据）。三者交融在行文中，不机械分段。

[末段：价值收束，60-100字]
不回扣论点编号做机械总结。把论证抬高一个层次，让听众感受到"这个持方意味着什么"。做实，不强行金句收束。
谢谢。
```

**格式硬要求**：

| 要求 | 说明 |
|------|------|
| **定义先行** | 开头即框定辩题核心词的含义，用理论/思想家/语境带入，忌照搬字典 |
| **判准前置** | 判准在论点之前明确写出，不可埋在行文中 |
| **一级论点 = 2 个** | 两个论点，每个以完整段落呈现，说理与证据交融 |
| **引用自然嵌入行文** | "黑格尔说……""正如……"——学术引用融入口语节奏，忌括号硬塞 |
| **论点开头直标"第一""第二"** | 论点段首句即亮出"第一/第二，……"，不埋在段落中间，评委一耳朵就能抓到 |
| **论点段落三层一体** | 每个论点段落内：主张（判断）+ 说理（推理）+ 举证（例子/数据），三者不机械分段 |
| **结尾做实不收束** | 不回扣论点编号做机械总结，不做强行金句——扎实收束论证即可 |
| **小明不出稿** | 小明是内部框架工具，仅在破题阶段用于锚定立场与共情方向 |
| **字数** | 全文 900-1100 字（约3分钟口播） |

**参考范例**（节选自第十届新国辩南大正一申论，辩题"当代年轻人应该活得更现实/更理想"）：

> 谢谢主席，各位好。今天我们要讨论的问题是理想与现实之争。也许在大家的认知中，理想积极而崇高，现实则消极、平淡乃至无意义，但这实属我们的误会。黑格尔说，现实比一切抽象的观念都更为深刻，现实即是一切事实的总和，它既蕴含着历史的沉淀，又潜藏着未来的潜能。理想则是某种观念或意识形态的外化。小时候，我们常说我的理想是成为科学家——哪怕那时候我们根本不知道科学是什么，科学家如何生活实践。但无需任何现实，理想依然成立，因为理想的实质就是某种意识形态所规划出的秩序等级：哪些是好，哪些不好，哪些先进，哪些落后。
>
> 而活得现实则是一切从现实出发。它在历史中的名称叫做实事求是。不是无意识地臣服于某些意识形态，而是试图解除这些遮蔽，将现实作为开发自己生活的原则。原来我们未必要30岁之前有车有房有存款，并且为此泣血拼搏；也可以认真而严肃地叩问自己的内心，从自身的感受出发，挖掘属于自己的答案。把自己作为方法，我们才可以解除理想的遮蔽。
>
> 最后，关于当代——这是一个东方与西方、新潮与旧派，各类价值观念与意识形态无孔不入又疯狂交织的混乱年代。未经审视的崇高背后是枷锁，不经反思的远方背后是谎言。当代年轻人的痛苦和无奈，往往来源于对理性权威的服从。理想不该成为定义人们生活的方式，年轻人更应该从现实中体悟反思，寻找属于自己的道路。这方是一种自我掌控的人生。

> **分析**：全稿约650字（1分42秒的片段），没有"首先/其次/最后"的显式分隔，没有括号引注，黑格尔直接入行文。定义、论点、价值三者以一段流水式的口播连贯推进——这就是高水准一辩稿的质地。

### 6.3 模块②：参考资料卡格式

**表格模板**：

```
| 编号 | 适配论点 | 核心实证/案例 | 上场话术 | 对方可能的反拆 → 我方反拆 |
|------|----------|---------------|----------|---------------------------|
| 01 | 论点一：XXX | 【机理/数据/案例简述】… | "……"（口播体） | 对方可能说"……" → 我方回："……" |
| 02 | 论点二：XXX | 【机理/数据/案例简述】… | "……"（口播体） | 对方可能说"……" → 我方回："……" |
| … | 底线防守 | 【逻辑闭环/价值兜底】… | "……"（口播体） | 最坏情况：对方质疑"……" → 我方兜底："……" |
```

**每张卡的内容拆解**：

- **核心实证/案例**：一句话讲清楚这个证据怎么支撑论点——是机理就写推演链，是案例就写"谁+做了什么+为什么算我方立场"。忌长篇堆砌。
- **上场话术**：直接把场上可以说的完整句子写出来，口播体，用引号包裹。这是你拿起来就能念的东西。
- **对方可能的反拆 → 我方反拆**：预判对方会怎么攻击这个例子，给一条最贴合的回应。**不求穷尽所有可能，只给最直接、最能一击封喉的那一条。**

**硬要求**：

| 要求 | 说明 |
|------|------|
| 每张卡精简 | 场上没时间翻长篇，每列都必须是一眼能扫完的信息量 |
| 话术必须口播体 | 直接可念，不要"可以说……"的提示语 |
| 引用完整可核验 | 作者+年份+来源，备赛期用 |
| 反拆只给一条 | 给出最贴合该例子的、最直接的回应，不堆多条 |
| 至少覆盖每个论点 1 张 | 底线卡至少 1 张 |

---

### 6.4 模块③：质询问题组格式

**结构模板**：

```
第X组：[战场名称]（针对[对方某论点/某前提]）

1. [确认问题1]（确认对方前提/定义/事实）
2. [追问1]（逼出逻辑漏洞）
3. [追问2]（收网——确认矛盾或荒谬结论）
4. [（可选）兜底问题]

【设计意图】：[这组问题瞄准的击穿点]
【对方可能恶逃方式与应对】：[归谬型/派别型/弃攻型/自洽型] → [我方应对]
```

**分组原则**：

| 战场 | 组数 | 目标 |
|------|------|------|
| **定义/讨论范围战场** | 1-2组 | 抢占讨论入口 |
| **判准战场** | 1-2组 | 攻击对方判准的合理性/可辩性/启发性 |
| **论点战场** | 每个对方论点 1-2组 | 逐点击穿 |
| **价值战场** | 1组 | 追问"你的方案对那个付出不被回应的人意味着什么" |
| **兜底组** | 1组 | 万金油问题，任何环节都能用 |

**硬要求**：

| 要求 | 说明 |
|------|------|
| 每组 3-4 题 | 太多质询超时，太少形不成追打 |
| 每题 ≤ 15字 | 质询问题必须短，让对方答，你不替他答 |
| 必须有设计意图 | 让使用者知道这组问题的"击穿点"在哪 |
| 必须有恶逃应对 | 预判对方四类恶逃（归谬/派别/弃攻/自洽），提前备好话术 |
| 优先确认式问题 | "你的论点是……对吧？"（确认），而非"你是不是觉得……？"（诱导） |

---

### 6.5 模块④：预设拆论格式

**结构模板**：

```
【对方可能论点X：[对方论点一句话概述]】

对方论证逻辑：[对方会怎么论证这个点，2-3句概括其推理链]

我方拆解思路：
1. [拆解点1]：[拆什么——定义/逻辑/论据/价值？怎么拆？]
2. [拆解点2]：[同上]
3. [（可选）拆解点3]

定列比结话术：
- 定：[明确反驳对象]
- 列：[一句话复述对方] vs [一句话亮出我方]
- 比：[为什么我更好]
- 结：[这与持方的关系]
```

**预判覆盖清单**（必须覆盖的对方论点类型）：

| 必预判类型 | 说明 |
|-----------|------|
| 对方核心论点 | 对方立论最可能打的1-2个一级论点 |
| 对方对我方核心论点的反击 | 对方必然预判我方的预判 |
| 对方的价值攻击 | "你方的主张太……（苛责/理想化/冷漠）" |
| 对方的替代方案 | "不需要X，用Y也可以" |

**硬要求**：

| 要求 | 说明 |
|------|------|
| 每个对方论点必须带定列比结 | 不能用"我觉得他说得不对"收尾 |
| 先列对方逻辑再拆 | 忌"还没听懂就开打"——先复述对方逻辑，确认理解正确 |
| 区分拆解点类型 | 标注是"拆定义""拆逻辑""拆论据"还是"拆价值" |

---

### 6.6 模块⑤：接质预测与回应格式

**结构模板**：

```
【质询预测X：[对方可能质询方向]】

对方质询：[模拟对方连续追问，2-3轮]
→ 我方回应：[回答 + 省略/缩短/后置让步 + 答"超"所问]

核心防守点：[这个质询在撬动我方哪个论证要素？]
```

**预判覆盖**：

| 必须预判的方向 | 说明 |
|---------------|------|
| 攻击我方定义/讨论范围 | 对方会试图压缩/拉偏讨论范围 |
| 攻击我方判准 | "你方判准不合理/不具可辩性" |
| 攻击我方核心论据 | 质疑数据真实性/相关性/充分性 |
| 攻击我方价值 | "你方价值太空/不切实际/伤害那些付出不被回应的人" |
| 攻击我方逻辑链 | "你的前提推不出结论" |

**接质六项技巧的格式嵌入**（引用自 §2.2.2）：

| 技巧 | 在回应中的标注方式 |
|------|-------------------|
| 省略/缩短/后置让步 | 在回应中用【让步】标记，如"【让步】承认X，但Y才是关键……" |
| 答"超"所问 | 用【升维】标记，如"【升维】这不是X问题，核心是Y问题……" |
| 用例子代替逻辑 | 用【举例】标记，后接具体场景 |
| 警惕诱导发问 | 用【识别】标记对方问题中隐藏的未经确认的前提 |

### 6.8 模块⑦：结辩稿格式

**结构模板**：

```
[第一段：争议梳理]（直面核心分歧，不要回避劣势）
"今天这场比赛，我们真正在争的是什么？我们争的是……对方辩友给出的方案是……但这其实是……"

[第二段：框架强化]（重申我方定义-判准-论点闭环，1-2段）
"各位请想一想……[回扣最核心的一个论据或案例]"
"今天我方的逻辑其实很简单：……[一句话收束全场比赛]"

[第三段：价值升华]（回到我方持方所代表的人群，情感打动）
"最后让我回到[那个在关系中持续付出却得不到回应的人]……"
"我们主张XX，不是在……而是在……[纠正对方贴的标签]"

[末句：做实收束，不强行金句]
```

**硬要求**：

| 要求 | 说明 |
|------|------|
| 直面核心争议 | 对方追打什么，结辩就回应什么——不能假装没发生过 |
| 淡化逻辑/事实 | 结辩不是二辩，不要重新引数据、拆逻辑 |
| 回归价值与情感 | "幽默是最好的防守，感动是最好的进攻" |
| 不要重复队友 | 不做"我方二辩举证的XX数据对方没回应"的技术性归纳 |
| 纠正标签 | 如果对方整场在给我方贴标签（"你们太苛责""你们太冷漠"），结辩必须摘掉 |
| 结尾做实不收束 | 结辩同样不需要强行金句——扎实收束即可 |
| 字数 | 500-800字（约2-2.5分钟口播） |

### 6.9 输出原则

**核心原则：按需给模块，不要默认全量输出。** 用户要什么就只给什么，不要擅自附带其他模块。

| 用户请求 | 输出 |
|----------|------|
| "写一辩稿" | **仅①**，不附带参考资料卡 |
| "准备质询" | **仅③**，不附带接质回应 |
| "准备接质" | **仅⑤** |
| "准备反驳" | **仅④** |
| "写结辩" | **仅⑦** |
| "写二辩申论稿" | **仅⑥** |
| "一辩稿+资料卡" 或明确提到两个模块 | 用户点了几个给几个 |
| "全套/订制包" 或 "全部" | ①→⑦ 全部7个模块 |
| "破题分析" | 新剑宗四问框架 + 价值阶梯（头脑风暴，不进入订制包格式） |

当用户请求模糊、不确定需要哪个模块时，先问一句再产出。
